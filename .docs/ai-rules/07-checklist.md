## 代码规范（强制检查）

### PSR-1 基础编码标准

- [ ] 纯 PHP 文件使用 `<?php` 标签
- [ ] **纯 PHP 文件未使用结束标签 `?>`**（强制要求）
- [ ] 文件使用 UTF-8 编码（无 BOM）
- [ ] 文件只声明符号或产生副作用，不混合

### PSR-12 扩展编码风格指南

- [ ] 使用 4 个空格缩进（不使用 Tab）
- [ ] 行尾无尾随空格
- [ ] 文件以空行结尾
- [ ] `declare(strict_types=1);` 在文件第一行
- [ ] 类和方法的左花括号在下一行
- [ ] 控制结构的左花括号在同一行
- [ ] 命名空间声明后有空行
- [ ] `use` 声明在命名空间声明之后

### 类型声明和现代特性

- [ ] PHP 代码符合 PSR-12 编码规范
- [ ] 使用类型声明（参数类型、返回类型）
- [ ] 代码示例完整可运行
- [ ] 使用 PHP 8.2+ 语法特性
- [ ] 优先使用现代特性（构造器属性提升、枚举、只读属性等）
