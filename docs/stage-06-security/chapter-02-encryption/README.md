# 6.2 密码与加密安全

## 目标

- 掌握 `password_hash` 和 `password_verify` 的使用。
- 理解 Argon2id 参数的最佳实践。
- 熟悉 Sodium 密码库的使用（对称/非对称加密、签名）。
- 了解敏感数据脱敏的方法。

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、代码示例和最佳实践：

1. **[密码加密](section-01-password-encryption.md)**：`password_hash`、`password_verify`、Argon2id 配置、密码策略、完整示例。

2. **[数据加密](section-02-data-encryption.md)**：Sodium 库、对称加密、非对称加密、数字签名、完整示例。

3. **[加密最佳实践](section-03-best-practices.md)**：密钥管理、敏感数据脱敏、加密存储、安全传输、完整示例。

## 核心概念

- **密码哈希**：单向加密，不可逆
- **数据加密**：双向加密，可解密
- **Sodium**：现代密码库
- **密钥管理**：安全存储密钥

## 学习建议

1. **重点掌握**：
   - 密码哈希的使用
   - Sodium 加密库
   - 密钥管理

2. **实践练习**：
   - 完成每小节后的练习题目
   - 实现加密系统
   - 处理敏感数据

## 完成本章后

- 能够安全地处理密码和数据加密。
- 理解加密算法，能够选择合适的方案。
- 掌握密钥管理，能够安全存储密钥。
- 具备构建生产级加密系统的能力。
