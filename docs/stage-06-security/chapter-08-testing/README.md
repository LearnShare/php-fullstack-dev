# 6.8 测试体系建设

## 目标

- 理解测试金字塔的概念。
- 掌握 PHPUnit 和 Pest 的使用。
- 了解单元测试、集成测试、功能测试的区别。
- 熟悉 Mock、Stub、Fakes 的使用。

## 章节内容

本章分为四个独立小节，每节提供详细的概念解释、代码示例和最佳实践：

1. **[测试金字塔](section-01-testing-pyramid.md)**：测试金字塔概念、测试类型、测试策略、完整示例。

2. **[单元测试](section-02-unit-testing.md)**：PHPUnit 基础、测试编写、断言使用、测试组织、完整示例。

3. **[集成测试](section-03-integration-testing.md)**：集成测试概念、数据库测试、API 测试、测试数据管理、完整示例。

4. **[E2E 测试](section-04-e2e-testing.md)**：端到端测试、浏览器测试、E2E 测试工具、完整示例。

## 核心概念

- **测试金字塔**：测试层次结构
- **单元测试**：测试单个函数/方法
- **集成测试**：测试组件交互
- **E2E 测试**：测试完整系统

## 学习建议

1. **重点掌握**：
   - PHPUnit 使用
   - 测试编写
   - 测试策略

2. **实践练习**：
   - 完成每小节后的练习题目
   - 编写测试用例
   - 建立测试体系

## 完成本章后

- 能够建立完整的测试体系。
- 理解测试金字塔，能够编写各类测试。
- 掌握测试工具，能够提升测试效率。
- 具备构建生产级测试系统的能力。
