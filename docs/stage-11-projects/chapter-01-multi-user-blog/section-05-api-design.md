## 常见问题

### Q: 如何设计 RESTful API？

A: 使用资源名词、HTTP 方法表示操作、使用状态码表示结果、保持接口一致性。

### Q: API 版本如何管理？

A: 可以通过 URL 路径、请求头、查询参数等方式管理版本。
