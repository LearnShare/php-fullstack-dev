## 基本用法

### 设计模式示例

```php
<?php
declare(strict_types=1);

// 设计模式提供可重用的解决方案
// 例如：工厂模式创建对象
interface Product {
    public function getName(): string;
}

class ProductFactory {
    public static function create(string $type): Product {
        return match($type) {
            'A' => new ProductA(),
            'B' => new ProductB(),
            default => throw new InvalidArgumentException()
        };
    }
}
```
