## 基本用法

### 模式实践示例

```php
<?php
declare(strict_types=1);

// 组合使用多个模式
class ServiceFactory {
    public function create(string $type): Service {
        // 工厂模式创建服务
        $service = match($type) {
            'A' => new ServiceA(),
            'B' => new ServiceB()
        };
        
        // 装饰器模式增强功能
        return new LoggingDecorator($service);
    }
}
```
