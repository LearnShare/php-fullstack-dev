# 7.4 异常体系与框架级设计

## 目标

- 理解异常体系的设计原则，能够设计合理的异常层次。

- **异常体系**：异常类的层次结构

- **框架异常**：框架级别的异常处理
- **异常处理**：统一的异常处理机制
