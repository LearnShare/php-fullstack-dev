# 4.3 超全局变量：Web 输入的核心

## 目标

- 深入理解 `$_GET`、`$_POST`、`$_SERVER`、`$_COOKIE`、`$_SESSION`、`$_FILES` 等超全局变量的使用。
- 掌握表单数据处理、查询参数解析、Header 信息获取、文件上传处理。
- 理解 `$_REQUEST` 的安全隐患，避免使用。
- 掌握输入过滤与验证，确保数据安全。

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、代码示例和最佳实践：

1. **[$_GET、$_POST 与 $_REQUEST](section-01-get-post-request.md)**：查询参数处理、表单数据处理、数组参数、`$_REQUEST` 安全隐患、输入过滤与验证及完整示例。

2. **[$_SERVER、$_SESSION 与 $_COOKIE](section-02-server-session-cookie.md)**：服务器信息获取、Header 处理、Session 管理、Cookie 安全设置、客户端 IP 获取及完整示例。

3. **[$_FILES 与安全处理](section-03-files-security.md)**：文件上传基础、文件验证、安全存储、文件类型检测、大小限制、安全最佳实践及完整示例。

## 核心概念

- **超全局变量**：在脚本任何位置都可访问的预定义变量
- **输入验证**：验证和清理用户输入，防止安全漏洞
- **数据来源**：明确数据来源（GET/POST/COOKIE），避免使用 `$_REQUEST`
- **安全转义**：输出时转义特殊字符，防止 XSS 攻击
