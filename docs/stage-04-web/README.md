# 阶段四：Web 服务与 API-First 开发

本阶段深入讲解 PHP 在 Web 开发中的核心应用，从 HTTP 请求响应流程开始，逐步掌握 HTML 渲染、表单处理、文件上传、RESTful API 设计、会话管理、鉴权授权等 Web 开发必备技能。每章提供完整示例、安全最佳实践与常见陷阱说明，帮助你构建安全、可扩展的 Web 应用。

1. [`chapter-01-request-response`](chapter-01-request-response/README.md)：Web 交互基础：从请求到响应。
2. [`chapter-02-html-rendering`](chapter-02-html-rendering/README.md)：HTML 渲染与输出控制。
3. [`chapter-03-superglobals`](chapter-03-superglobals/README.md)：超全局变量：Web 输入的核心。
4. [`chapter-04-json-requests`](chapter-04-json-requests/README.md)：请求体解析：处理 JSON / API 请求。
5. [`chapter-05-file-upload`](chapter-05-file-upload/README.md)：文件上传处理。
6. [`chapter-06-restful-api`](chapter-06-restful-api/README.md)：RESTful API 设计与接口规范。
7. [`chapter-07-response-cors`](chapter-07-response-cors/README.md)：响应处理与跨域（CORS）。
8. [`chapter-08-session`](chapter-08-session/README.md)：会话与状态管理。
9. [`chapter-09-auth`](chapter-09-auth/README.md)：鉴权与授权模型（AuthN & AuthZ）。
10. [`chapter-10-traffic-security`](chapter-10-traffic-security/README.md)：流量治理与安全。
11. [`chapter-11-http-client`](chapter-11-http-client/README.md)：HTTP 客户端：Guzzle 详细使用。

完成阶段四后，你将具备：

- 深入理解 HTTP 协议与 Web 请求响应流程，能够处理各种类型的 Web 请求。
- 掌握 HTML 渲染、模板引擎使用、输出缓冲等 Web 输出技术。
- 熟练使用超全局变量处理表单数据、查询参数、文件上传等 Web 输入。
- 能够设计符合 RESTful 规范的 API，处理 JSON 请求响应。
- 理解会话管理、Cookie、Session、JWT 等状态管理机制。
- 掌握鉴权授权、OAuth2、RBAC 等安全机制。
- 具备 API 安全防护能力：CORS、Rate Limiting、请求签名等。
