# 9.3 Laravel（2025 最新特性）

## 目标

- 掌握 Laravel 框架的核心概念和使用方法。
- 理解 Laravel 的核心组件和架构设计。
- 能够使用 Laravel 开发完整的 Web 应用。
- 了解 Laravel 2025 最新特性。

## 章节内容

本章分为五个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[9.3.1 Laravel 基础](section-01-laravel-basics.md)**：Laravel 概述、安装配置、项目结构、基础使用、完整示例。

2. **[9.3.2 核心组件](section-02-core-components.md)**：服务容器、服务提供者、门面、契约、完整示例。

3. **[9.3.3 数据库与 ORM](section-03-database-orm.md)**：数据库配置、Eloquent ORM、查询构建器、迁移、完整示例。

4. **[9.3.4 认证与授权](section-04-auth.md)**：认证系统、授权系统、中间件、策略、完整示例。

5. **[9.3.5 2025 最新特性](section-05-latest-features.md)**：Laravel 11+ 新特性、性能改进、新功能、最佳实践、完整示例。

## 核心概念

- **Laravel**：流行的 PHP Web 框架
- **Eloquent ORM**：Laravel 的 ORM 实现
- **服务容器**：Laravel 的 IoC 容器
- **服务提供者**：注册和配置服务的组件

## 学习建议

1. **按顺序学习**：按顺序学习五个小节，理解 Laravel 的完整体系。

2. **重点掌握**：
   - Laravel 的核心概念
   - 核心组件的使用
   - 数据库和 ORM 的使用
   - 认证和授权

3. **实践练习**：
   - 完成每小节后的练习题目
   - 创建 Laravel 项目
   - 实现完整的 Web 应用

## 完成本章后

- 能够使用 Laravel 开发 Web 应用。
- 能够理解 Laravel 的核心组件。
- 能够使用 Eloquent ORM 操作数据库。
- 能够实现认证和授权功能。
- 了解 Laravel 2025 最新特性。
