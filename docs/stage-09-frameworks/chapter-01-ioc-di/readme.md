## 目标

- 理解 IoC（控制反转）和 DI（依赖注入）的核心概念和原理。
- 掌握 IoC 容器的实现原理和使用方法。

## 核心概念

- **IoC（控制反转）**：将对象的创建和管理交给容器
- **DI（依赖注入）**：通过构造函数、方法或属性注入依赖
- **IoC 容器**：管理对象生命周期和依赖关系的容器
- **服务提供者**：注册和配置服务的组件

## 学习建议

1. **按顺序学习**：按顺序学习三个小节，理解 IoC/DI 的完整体系。

2. **重点掌握**：

   - 服务提供者的使用

3. **实践练习**：
   - 完成每小节后的练习题目
   - 实现简单的 IoC 容器
   - 实现依赖注入
