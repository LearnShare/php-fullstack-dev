## 常见问题

### Q: Phalcon 为什么性能好？

A: Phalcon 用 C 扩展实现，减少了 PHP 解释器的开销，性能更高。

### Q: Phalcon 的安装有什么要求？

A: 需要编译 C 扩展，需要 root 权限，需要匹配的 PHP 版本。
