# 5.8 响应处理与跨域（CORS）

## 目标

- 掌握 HTTP 响应的处理方法，能够正确设置响应头和状态码。
- 理解 CORS 跨域机制，能够处理跨域请求。
- 掌握响应格式的统一方法，能够提供一致的 API 响应。

## 章节内容

本章分为两个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[5.8.1 响应处理](section-01-response-handling.md)**：HTTP 响应概述、响应头设置、状态码设置、响应体格式、JSON 响应、完整示例。

2. **[5.8.2 CORS 跨域处理](section-02-cors.md)**：CORS 概念、同源策略、CORS 请求流程、响应头设置、预检请求、完整示例。

## 核心概念

- **HTTP 响应**：服务器返回给客户端的数据
- **响应头**：响应的元数据信息
- **状态码**：响应的状态标识
- **CORS**：跨域资源共享机制

## 学习建议

1. **按顺序学习**：按顺序学习两个小节，理解响应处理和跨域的完整体系。

2. **重点掌握**：
   - 响应头设置
   - 状态码使用
   - CORS 配置
   - 预检请求处理

3. **实践练习**：
   - 完成每小节后的练习题目
   - 实现 API 响应处理
   - 配置 CORS 跨域

## 完成本章后

- 能够设置响应头和状态码。
