## 基本用法

### 验证示例

```php
<?php
declare(strict_types=1);

function validateUserInput(array $data): array {
    $errors = [];
    
    if (empty($data['email'])) {
        $errors[] = 'Email 是必填项';
    } elseif (!filter_var($data['email'], FILTER_VALIDATE_EMAIL)) {
        $errors[] = 'Email 格式不正确';
    }
    
    return $errors;
}
```
