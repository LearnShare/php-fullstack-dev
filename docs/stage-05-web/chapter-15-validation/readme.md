# 5.15 请求验证与数据校验

## 目标

- 理解请求验证的重要性，掌握数据验证的基本方法。
- 掌握 Symfony Validator 和 Respect/Validation 等验证库的使用。
- 能够编写自定义验证规则，实现灵活的验证系统。

## 章节内容

本章分为四个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[5.15.1 请求验证概述](section-01-overview.md)**：验证概念、验证的重要性、验证时机、验证规则、完整示例。

2. **[5.15.2 Symfony Validator](section-02-symfony-validator.md)**：Symfony Validator 概述、安装配置、验证规则、验证执行、错误处理、完整示例。

3. **[5.15.3 Respect/Validation](section-03-respect-validation.md)**：Respect/Validation 概述、安装配置、验证规则、链式验证、自定义规则、完整示例。

4. **[5.15.4 自定义验证规则](section-04-custom-rules.md)**：自定义规则概述、规则编写、规则注册、规则复用、完整示例。

## 核心概念

- **数据验证**：确保数据符合要求的过程
- **验证规则**：数据必须满足的条件
- **验证库**：提供验证功能的工具
- **自定义规则**：特定业务需求的验证规则

## 学习建议

1. **按顺序学习**：按顺序学习四个小节，理解请求验证的完整体系。

2. **重点掌握**：
   - 验证的重要性
   - 验证库的使用
   - 自定义规则编写
   - 验证最佳实践

3. **实践练习**：
   - 完成每小节后的练习题目
   - 实现验证功能
   - 理解验证的应用

## 完成本章后

- 能够进行请求数据验证。
- 能够使用验证库简化验证。
- 能够编写自定义验证规则。
- 能够构建健壮的验证系统。
