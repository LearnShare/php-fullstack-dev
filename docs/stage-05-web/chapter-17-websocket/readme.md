# 5.17 WebSocket 服务器

## 目标

- 理解 WebSocket 协议的特点和应用场景，掌握 WebSocket 服务器的实现方法。
- 掌握 Ratchet 和 ReactPHP WebSocket 的使用，能够实现实时通信功能。

## 核心概念

- **WebSocket**：全双工通信协议
- **实时通信**：即时消息传递
- **长连接**：持久连接
- **事件驱动**：基于事件的编程模型
