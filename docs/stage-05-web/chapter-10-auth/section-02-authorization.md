## 基本用法

### 权限检查示例

```php
<?php
declare(strict_types=1);

function checkPermission(string $permission): bool {
    $user = getCurrentUser();
    return $user->hasPermission($permission);
}

// 使用
if (checkPermission('edit_post')) {
    // 允许编辑
}
```
