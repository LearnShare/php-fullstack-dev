# 5.12 HTTP 客户端

## 目标

- 掌握 Guzzle HTTP 客户端库的使用，能够进行 HTTP 请求。
- 理解异步请求的概念，能够实现异步 HTTP 请求。
- 掌握错误处理和重试机制，能够构建健壮的 HTTP 客户端。

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[5.12.1 Guzzle 基础](section-01-guzzle-basics.md)**：Guzzle 概述、安装配置、基本请求、请求选项、响应处理、完整示例。

2. **[5.12.2 异步请求](section-02-async-requests.md)**：异步请求概念、Promise、并发请求、异步处理、完整示例。

3. **[5.12.3 错误处理与重试](section-03-error-handling-retry.md)**：错误处理、异常类型、重试机制、重试策略、完整示例。

## 核心概念

- **Guzzle**：PHP HTTP 客户端库
- **异步请求**：非阻塞的 HTTP 请求
- **Promise**：异步操作的表示
- **重试机制**：请求失败后的重试策略

## 学习建议

1. **按顺序学习**：按顺序学习三个小节，理解 HTTP 客户端的完整体系。

2. **重点掌握**：
   - Guzzle 基本使用
   - 异步请求处理
   - 错误处理和重试

3. **实践练习**：
   - 完成每小节后的练习题目
   - 实现 HTTP 客户端功能
   - 理解异步请求的优势

## 完成本章后

- 能够使用 Guzzle 进行 HTTP 请求。
- 能够实现异步 HTTP 请求。
- 能够处理错误和实现重试机制。
- 能够构建健壮的 HTTP 客户端。
