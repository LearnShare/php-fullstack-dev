## 基本用法

### 动态路由示例

```php
<?php
declare(strict_types=1);

// 动态路由定义
Route::get('/users/{id}', function($id) {
    return getUser($id);
});

// 带约束的路由
Route::get('/users/{id}', function($id) {
    // id 必须是数字
})->where('id', '[0-9]+');
```
