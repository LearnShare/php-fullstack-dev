## 基本用法

### 路由组织示例

```php
<?php
declare(strict_types=1);

// 路由分组
Route::group(['prefix' => 'api/v1', 'middleware' => 'auth'], function() {
    Route::get('/users', 'UserController@index');
    Route::post('/users', 'UserController@store');
});
```
