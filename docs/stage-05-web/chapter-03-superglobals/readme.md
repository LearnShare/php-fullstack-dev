# 5.3 超全局变量：Web 输入的核心

## 目标

- 掌握 PHP 中超全局变量的使用，理解它们在 Web 开发中的核心作用。
- 掌握 $_GET、$_POST、$_REQUEST 的区别和使用场景。
- 掌握 $_SERVER、$_SESSION、$_COOKIE 的使用方法。
- 掌握 $_FILES 的文件上传处理和安全注意事项。

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[5.3.1 $_GET、$_POST 与 $_REQUEST](section-01-get-post-request.md)**：$_GET 超全局变量、$_POST 超全局变量、$_REQUEST 超全局变量、数据获取、数据验证、完整示例。

3. **[5.3.3 $_FILES 与安全处理](section-03-files-security.md)**：$_FILES 超全局变量、文件上传处理、文件验证、安全注意事项、完整示例。

## 核心概念

- **超全局变量**：在任何作用域都可访问的变量
- **$_GET**：URL 参数数据
- **$_POST**：POST 请求体数据
- **$_SESSION**：会话数据
- **$_COOKIE**：Cookie 数据
- **$_FILES**：上传文件数据
