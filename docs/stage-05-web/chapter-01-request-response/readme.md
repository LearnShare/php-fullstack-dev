# 5.1 Web 交互基础：从请求到响应

## 目标

- 理解 HTTP 请求和响应的完整流程，掌握 Web 应用的基本工作原理。
- 理解 Web Server 与 PHP-FPM 的协作机制，掌握 Web 架构的基础知识。
- 理解 Shared Nothing 架构的概念和优势，掌握 PHP Web 应用的特点。

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[5.1.1 HTTP 请求响应流程](section-01-http-flow.md)**：HTTP 协议基础、请求结构、响应结构、请求生命周期、完整示例。

2. **[5.1.2 Web Server 与 PHP-FPM 协作](section-02-web-server-fpm.md)**：Web Server 作用、PHP-FPM 作用、FastCGI 协议、协作流程、配置优化、完整示例。

3. **[5.1.3 Shared Nothing 架构](section-03-shared-nothing.md)**：Shared Nothing 概念、PHP 的 Shared Nothing 特性、无状态设计、扩展性优势、完整示例。

## 核心概念

- **HTTP 协议**：Web 通信的基础协议
- **请求-响应模型**：Web 交互的基本模式
- **PHP-FPM**：PHP FastCGI 进程管理器
- **Shared Nothing**：无共享架构设计
