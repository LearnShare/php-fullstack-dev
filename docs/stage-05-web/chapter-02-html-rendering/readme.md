# 5.2 HTML 渲染与输出控制

## 目标

- 掌握 HTML 渲染的基本方法，理解模板引擎的作用和使用。
- 掌握输出缓冲机制，能够控制输出的时机和内容。
- 理解模板引擎的工作原理，能够选择合适的模板引擎。

## 章节内容

本章分为两个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[5.2.1 HTML 渲染与模板引擎](section-01-rendering-templates.md)**：HTML 渲染方法、模板引擎概念、Twig、Blade、原生 PHP 模板、模板继承、完整示例。

2. **[5.2.2 输出缓冲与控制](section-02-output-buffering.md)**：输出缓冲概念、ob_start()、ob_get_clean()、输出控制函数、完整示例。

## 核心概念

- **模板引擎**：分离逻辑和视图的工具
- **输出缓冲**：控制输出时机的机制
- **模板继承**：模板复用的方法
- **视图分离**：MVC 架构的重要组成部分

## 学习建议

1. **按顺序学习**：按顺序学习两个小节，理解 HTML 渲染和输出控制的完整体系。

2. **重点掌握**：
   - HTML 渲染方法
   - 模板引擎使用
   - 输出缓冲机制

3. **实践练习**：
   - 完成每小节后的练习题目
   - 实现简单的模板系统
   - 理解输出缓冲的应用

## 完成本章后

- 能够进行 HTML 渲染。
- 能够使用模板引擎。
- 能够控制输出缓冲。
- 能够实现视图分离。
