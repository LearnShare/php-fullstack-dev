## 基本用法

### 原生 PHP 模板示例

```php
<?php
declare(strict_types=1);

$title = '页面标题';
$content = '页面内容';
include 'template.php';
```

### Twig 示例

```twig
{% extends "base.html" %}
{% block content %}
    <h1>{{ title }}</h1>
    <p>{{ content }}</p>
{% endblock %}
```
