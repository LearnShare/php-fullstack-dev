# 3.1 类、对象与基础 OOP

## 目标

- 理解类（Class）与对象（Object）的基本概念，掌握如何定义类、创建对象、访问属性与方法。
- 熟悉可见性修饰符（`public`、`protected`、`private`）的作用与使用场景。
- 掌握构造函数（`__construct`）、析构函数（`__destruct`）与魔术方法的基础用法。
- 理解 `$this` 关键字、对象克隆（`clone`）与对象引用的区别。

## 核心概念

- **类（Class）**：对象的模板，定义对象的属性和方法
- **对象（Object）**：类的实例，拥有独立的属性值
- **封装（Encapsulation）**：通过可见性修饰符控制访问
- **生命周期**：构造函数初始化，析构函数清理

## 学习建议

1. **按顺序学习**：按顺序学习五个小节，理解类与对象的完整体系。

2. **重点掌握**：
   - 类与对象的基本概念
   - 可见性修饰符的使用
   - 构造函数和析构函数
   - 对象克隆和引用
   - 静态成员和魔术方法

3. **实践练习**：
   - 完成每小节后的练习题目
   - 创建实际的类并实例化对象
   - 理解封装的重要性

## 完成本章后

- 能够定义类、创建对象、访问属性和方法。

- 能够使用构造函数和析构函数管理对象生命周期。
- 能够理解对象克隆和引用的区别。
- 能够使用静态成员和魔术方法扩展对象功能。
