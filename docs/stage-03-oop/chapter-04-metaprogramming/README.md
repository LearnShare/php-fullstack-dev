# 3.4 代码模块化与元编程

## 目标

- 理解 Traits 的概念与使用场景，掌握如何通过 Traits 实现代码复用。
- 熟悉 Traits 的冲突解决机制，了解优先级与方法别名。
- 掌握 PHP 8.0+ Attributes（注解）系统，能够创建和使用自定义属性。
- 理解元编程的概念，能够在实际项目中应用 Traits 和 Attributes。

## 章节内容

本章分为两个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[Traits](section-01-traits.md)**：Traits 基础、多个 Traits、冲突解决、可见性修改、抽象方法、属性定义及完整示例。

2. **[Attributes（注解）](section-02-attributes.md)**：基础语法、读取 Attributes、Attribute 目标、自定义 Attribute 示例（验证器、路由系统）、与 Traits 结合使用及完整示例。

## 核心概念

- **Traits**：代码复用机制，解决单继承限制
- **Attributes**：元数据系统，提供类型安全的注解

## 学习建议

1. **按顺序学习**：按顺序学习两个小节，理解 Traits 和 Attributes 的使用。

2. **重点掌握**：
   - Traits 的定义和使用
   - Traits 冲突解决
   - Attributes 的定义和读取
   - 实际应用场景

3. **实践练习**：
   - 完成每小节后的练习题目
   - 使用 Traits 实现代码复用
   - 使用 Attributes 实现元编程功能

## 完成本章后

- 能够使用 Traits 实现代码复用，解决单继承限制。
- 能够解决 Traits 的方法冲突问题。
- 能够定义和使用 Attributes 添加元数据。
- 能够在实际项目中应用 Traits 和 Attributes。
