# 6.4 ORM 框架与数据迁移

## 目标

- 理解 ORM 的概念和优势，掌握 ORM 框架的使用方法。
- 掌握 Eloquent ORM 的使用，能够进行高效的数据操作。
- 掌握数据迁移的方法，能够管理数据库结构变更。
- 理解关系映射，能够处理复杂的数据库关系。

## 章节内容

本章分为四个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[6.4.1 ORM 基础](section-01-orm-basics.md)**：ORM 概念、ORM 的优势、ORM 框架对比、Active Record 模式、完整示例。

2. **[6.4.2 Eloquent 使用](section-02-eloquent.md)**：Eloquent 概述、模型定义、查询构建器、CRUD 操作、完整示例。

3. **[6.4.3 数据迁移](section-03-migrations.md)**：数据迁移概念、迁移文件创建、迁移执行、回滚、完整示例。

4. **[6.4.4 关系映射](section-04-relationships.md)**：关系类型（一对一、一对多、多对多）、关系定义、关系查询、完整示例。

## 核心概念

- **ORM**：对象关系映射
- **Active Record**：活动记录模式
- **数据迁移**：数据库结构版本管理
- **关系映射**：数据库关系的对象表示

## 学习建议

1. **按顺序学习**：按顺序学习四个小节，理解 ORM 的完整体系。

2. **重点掌握**：
   - ORM 概念和优势
   - Eloquent 使用
   - 数据迁移
   - 关系映射

3. **实践练习**：
   - 完成每小节后的练习题目
   - 使用 ORM 实现数据操作
   - 理解 ORM 的优势

## 完成本章后

- 能够理解 ORM 的概念和优势。
- 能够使用 Eloquent 进行数据操作。
- 能够使用数据迁移管理数据库结构。
- 能够处理复杂的数据库关系。
