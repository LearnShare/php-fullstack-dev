## 基本用法

### 性能指标监控示例

```sql
-- 查看连接状态
SHOW STATUS LIKE 'Threads_connected';
SHOW STATUS LIKE 'Max_used_connections';

-- 查看查询统计
SHOW STATUS LIKE 'Questions';
SHOW STATUS LIKE 'Com_select';

-- 查看缓存命中率
SHOW STATUS LIKE 'Qcache_hits';
```
