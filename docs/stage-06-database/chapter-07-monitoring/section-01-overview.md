## 基本用法

### 性能监控示例

```sql
-- 查看当前连接数
SHOW STATUS LIKE 'Threads_connected';

-- 查看查询统计
SHOW STATUS LIKE 'Questions';

-- 查看慢查询数量
SHOW STATUS LIKE 'Slow_queries';
```
