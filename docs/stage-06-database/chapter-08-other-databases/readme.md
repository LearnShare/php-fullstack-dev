# 6.8 其他数据库

## 目标

- 了解其他关系型数据库的特点和使用场景。
- 了解 NoSQL 数据库的概念和类型，理解 NoSQL 与关系型数据库的区别。
- 能够根据场景选择合适的数据库。

## 章节内容

本章分为两个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[6.8.1 其他关系型数据库](section-01-other-relational.md)**：PostgreSQL、SQLite、MariaDB、数据库对比、选择原则、完整示例。

2. **[6.8.2 NoSQL 数据库](section-02-nosql.md)**：NoSQL 概念、NoSQL 类型（文档型、键值型、列式、图型）、MongoDB、选择原则、完整示例。

## 核心概念

- **关系型数据库**：基于关系模型的数据库
- **NoSQL**：非关系型数据库
- **文档数据库**：以文档形式存储数据
- **键值数据库**：以键值对形式存储数据

## 学习建议

1. **按顺序学习**：按顺序学习两个小节，理解不同数据库的特点。

2. **重点掌握**：
   - 数据库类型对比
   - 使用场景选择
   - 数据库特点

3. **实践练习**：
   - 完成每小节后的练习题目
   - 理解不同数据库的适用场景
   - 掌握数据库选择原则

## 完成本章后

- 能够了解不同数据库的特点。
- 能够根据场景选择合适的数据库。
- 能够理解 NoSQL 数据库的应用。
- 能够进行数据库选型。
