# 6.2 PDO 入门与高安全模式

## 目标

- 掌握 PDO 的基础使用方法，能够进行数据库连接和操作。
- 理解预处理语句的重要性，能够防止 SQL 注入攻击。
- 掌握 PDO 的 CRUD 操作和高级特性，能够高效地进行数据库操作。

## 章节内容

本章分为四个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[6.2.1 PDO 基础与连接](section-01-pdo-basics.md)**：PDO 概述、PDO 连接、连接配置、错误处理、完整示例。

2. **[6.2.2 预处理语句与防注入](section-02-prepared-statements.md)**：预处理语句概念、预处理语句使用、参数绑定、防注入原理、完整示例。

3. **[6.2.3 CRUD 操作](section-03-crud-operations.md)**：创建（Create）、读取（Read）、更新（Update）、删除（Delete）、完整示例。

4. **[6.2.4 高级特性](section-04-advanced-features.md)**：事务处理、存储过程、批量操作、结果集处理、完整示例。

## 核心概念

- **PDO**：PHP Data Objects，PHP 数据对象
- **预处理语句**：预编译的 SQL 语句
- **参数绑定**：安全的参数传递方式
- **事务**：数据库操作的原子性保证

## 学习建议

1. **按顺序学习**：按顺序学习四个小节，理解 PDO 的完整体系。

2. **重点掌握**：
   - PDO 连接和配置
   - 预处理语句使用
   - CRUD 操作
   - 事务处理

3. **实践练习**：
   - 完成每小节后的练习题目
   - 实现数据库操作功能
   - 理解安全性的重要性

## 完成本章后

- 能够使用 PDO 进行数据库操作。
- 能够使用预处理语句防止 SQL 注入。
- 能够进行 CRUD 操作。
- 能够使用事务保证数据一致性。
