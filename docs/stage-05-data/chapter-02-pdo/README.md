# 5.2 PDO 入门与高安全模式

## 目标

- 掌握 PDO（PHP Data Objects）的基础用法。
- 理解 DSN 格式与连接选项配置。
- 深入理解预处理语句防注入的原理。
- 掌握 `bindParam` 与 `bindValue` 的区别。
- 能够进行安全的 CRUD 操作。

## 章节内容

本章分为四个独立小节，每节提供详细的概念解释、代码示例和最佳实践：

1. **[PDO 基础与连接](section-01-pdo-basics.md)**：什么是 PDO、DSN 格式、连接选项配置、错误处理模式及完整示例。

2. **[预处理语句与防注入](section-02-prepared-statements.md)**：SQL 注入原理、预处理语句、`bindParam` vs `bindValue`、防注入最佳实践及完整示例。

3. **[CRUD 操作](section-03-crud-operations.md)**：Create（创建）、Read（读取）、Update（更新）、Delete（删除）、获取模式、完整示例。

4. **[高级特性](section-04-advanced-features.md)**：事务处理、获取模式、错误处理、批量操作、性能优化及完整示例。

## 核心概念

- **PDO**：PHP 数据对象，统一的数据库访问接口
- **预处理语句**：防止 SQL 注入的关键机制
- **DSN**：数据源名称，连接字符串
- **获取模式**：数据获取的方式

## 学习建议

1. **重点掌握**：
   - 预处理语句的使用
   - SQL 注入防护
   - CRUD 操作

2. **实践练习**：
   - 完成每小节后的练习题目
   - 实现一个完整的数据库操作类
   - 处理各种数据库操作场景

## 完成本章后

- 能够安全地使用 PDO 进行数据库操作。
- 理解 SQL 注入原理，能够有效防护。
- 掌握 CRUD 操作，能够处理各种数据操作。
- 具备构建生产级数据库操作类的能力。
