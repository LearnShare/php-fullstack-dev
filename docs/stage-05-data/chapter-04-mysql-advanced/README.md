# 5.4 现代 MySQL 高级特性

## 目标

- 掌握 MySQL JSON 字段的使用，包括查询、索引和存储结构。
- 理解窗口函数（Window Functions）的应用场景。
- 熟悉 CTE（Common Table Expression）与递归查询。
- 掌握 MySQL 性能优化的高级技巧。

## 章节内容

本章分为四个独立小节，每节提供详细的概念解释、代码示例和最佳实践：

1. **[JSON 字段](section-01-json-fields.md)**：JSON 字段创建、插入、查询、更新、索引、JSON 函数及完整示例。

2. **[窗口函数](section-02-window-functions.md)**：窗口函数基础、排名函数、聚合窗口函数、窗口函数应用场景及完整示例。

3. **[CTE 与递归查询](section-03-cte-recursive.md)**：CTE 基础、递归 CTE、递归查询应用场景、性能优化及完整示例。

4. **[性能优化](section-04-performance.md)**：查询优化、索引优化、分区表、执行计划分析及完整示例。

## 核心概念

- **JSON 字段**：存储和查询 JSON 数据
- **窗口函数**：在结果集上执行计算
- **CTE**：公共表表达式，简化复杂查询
- **递归查询**：处理层次结构数据

## 学习建议

1. **重点掌握**：
   - JSON 字段的查询和索引
   - 窗口函数的使用
   - CTE 和递归查询

2. **实践练习**：
   - 完成每小节后的练习题目
   - 实现复杂的查询场景
   - 优化查询性能

- 能够使用 MySQL 的高级特性处理复杂场景。
- 理解窗口函数，能够优化查询性能。
