# 5.8 文件系统流式操作

## 目标

- 理解流（Stream）的概念与使用场景。
- 掌握 PHP 流式操作的基本用法。
- 熟悉大文件处理的方法。
- 了解文件上传和下载的流式处理。

## 章节内容

本章分为两个独立小节，每节提供详细的概念解释、代码示例和最佳实践：

1. **[流式操作](section-01-streaming.md)**：流的概念、流包装器、流上下文、流过滤器、完整示例。

2. **[大文件处理](section-02-large-files.md)**：大文件读取、大文件写入、大文件上传、大文件下载、内存优化、完整示例。

## 核心概念

- **流（Stream）**：数据流的抽象
- **流包装器**：访问不同数据源
- **流过滤器**：处理流数据
- **大文件处理**：内存友好的文件操作
