# 2.3.1 变量基础

## 概述

变量是存储数据的容器，是编程的基础概念。本节介绍 PHP 变量的声明、赋值、命名规则、变量作用域等基础知识，帮助理解 PHP 变量的动态特性。

**章节类型**：语法性章节

**主要内容**：
- 变量的声明和赋值
- 变量命名规则
- 变量的动态特性
- 变量作用域（局部、全局）
- 变量的类型
- 变量使用示例

## 特性

- **动态类型**：PHP 变量是动态类型的，不需要声明类型
- **弱类型**：变量类型可以自动转换
- **作用域**：变量有作用域限制

## 语法/定义

### 变量声明

**语法**：`$variable_name = value;`
**特点**：变量名以 `$` 开头
**示例**：`$name = "World";`

### 变量命名规则

- 以字母或下划线开头
- 只能包含字母、数字、下划线
- 区分大小写
- 不能使用 PHP 关键字

## 基本用法

### 基础示例

```php
<?php
declare(strict_types=1);

// 声明和赋值
$name = "World";
$age = 25;
$price = 99.99;

// 使用变量
echo "Hello, {$name}!\n";
echo "Age: {$age}\n";
echo "Price: {$price}\n";
```

### 变量作用域示例

```php
<?php
declare(strict_types=1);

$global = "Global variable";

function test(): void
{
    $local = "Local variable";
    echo $local . "\n";
    // echo $global;  // 错误：无法访问全局变量
}

test();
```

## 使用场景

- **数据存储**：存储各种类型的数据
- **计算**：存储计算结果
- **配置**：存储配置信息
- **状态**：存储程序状态

## 注意事项

- **变量名区分大小写**：`$name` 和 `$Name` 是不同的变量
- **作用域限制**：局部变量外部不可见
- **未定义变量**：访问未定义变量会触发 Notice

## 常见问题

### 问题 1：变量未定义
- **原因**：变量未声明就使用
- **解决**：在使用前声明变量

### 问题 2：作用域问题
- **原因**：在错误的作用域访问变量
- **解决**：理解变量作用域

## 最佳实践

- 使用有意义的变量名
- 在使用前声明变量
- 理解变量作用域
- 避免使用全局变量
