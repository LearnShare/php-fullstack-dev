## 核心概念

- **变量**：用于存储数据的容器，PHP 变量是动态类型的
- **引用**：变量别名，多个变量指向同一个值
- **可变变量**：变量名可以动态生成
- **常量**：不可改变的值，使用 `const` 或 `define()` 定义
- **作用域**：变量的可见范围
