# 2.15 时间与日期处理

## 目标

## 核心概念

- **DateTime**：可变对象，修改会改变原对象
- **DateTimeImmutable**：不可变对象，每次操作返回新实例（推荐）
- **DateTimeZone**：时区对象，用于处理不同时区的时间
- **DateInterval**：时间间隔对象，用于表示时间差
- **时间戳**：Unix 时间戳（秒）和微秒时间戳
