# 2.6.1 算术运算符

## 概述

算术运算符用于进行数学计算。本节介绍加法、减法、乘法、除法、取模、指数运算符的语法、用法、类型转换及完整示例。

**主要内容**：
- 加法运算符（`+`）
- 减法运算符（`-`）
- 乘法运算符（`*`）
- 除法运算符（`/`）
- 取模运算符（`%`）
- 指数运算符（`**`）
- 类型转换规则
- 常见陷阱和注意事项

## 特性

- **基本运算**：支持基本的数学运算
- **类型转换**：操作数会自动转换为数字
- **精度问题**：浮点数运算可能有精度问题

## 语法/定义

### 算术运算符

| 运算符 | 名称 | 示例 | 说明 |
| :----- | :--- | :--- | :--- |
| `+` | 加法 | `$a + $b` | 两数相加 |
| `-` | 减法 | `$a - $b` | 两数相减 |
| `*` | 乘法 | `$a * $b` | 两数相乘 |
| `/` | 除法 | `$a / $b` | 两数相除 |
| `%` | 取模 | `$a % $b` | 取余数 |
| `**` | 指数 | `$a ** $b` | 幂运算 |

## 基本用法

### 基础示例

```php
<?php
declare(strict_types=1);

$a = 10;
$b = 3;

echo $a + $b;  // 13
echo $a - $b;  // 7
echo $a * $b;  // 30
echo $a / $b;  // 3.333...
echo $a % $b;  // 1
echo $a ** $b; // 1000
```

### 类型转换示例

```php
<?php
declare(strict_types=1);

"10" + 5;   // 15，字符串转换为数字
"10.5" + 5; // 15.5，字符串转换为浮点数
```

## 注意事项

- **除零错误**：除以零会产生警告或错误
- **精度问题**：浮点数运算可能有精度问题
- **类型转换**：操作数会自动转换为数字

## 常见问题

### 问题 1：除零错误
- **原因**：除数为零
- **解决**：检查除数是否为零

### 问题 2：浮点数精度问题
- **原因**：浮点数精度有限
- **解决**：使用 `bcmath` 扩展进行精确计算

## 最佳实践

- 检查除零情况
- 注意浮点数精度问题
- 使用 `bcmath` 进行精确计算
