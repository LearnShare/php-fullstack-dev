# 2.6.2 赋值运算符

## 概述

赋值运算符用于给变量赋值。本节介绍基本赋值、复合赋值、自增自减、数组解构赋值（PHP 7.1+）的语法及使用场景。

**章节类型**：语法性章节

**主要内容**：
- 基本赋值（`=`）
- 复合赋值运算符（`+=`、`-=`、`*=`、`/=`、`%=`、`**=`、`.=`）
- 自增自减（`++`、`--`）
- 数组解构赋值（PHP 7.1+）
- 赋值运算符的优先级
- 使用场景和注意事项

## 特性

- **基本赋值**：将值赋给变量
- **复合赋值**：先运算再赋值
- **自增自减**：变量值加1或减1
- **数组解构**：从数组提取值赋给变量

## 语法/定义

### 基本赋值

**语法**：`$variable = value;`
**功能**：将值赋给变量

### 复合赋值运算符

| 运算符 | 等价于 | 说明 |
| :----- | :----- | :--- |
| `+=` | `$a = $a + $b` | 加法赋值 |
| `-=` | `$a = $a - $b` | 减法赋值 |
| `*=` | `$a = $a * $b` | 乘法赋值 |
| `/=` | `$a = $a / $b` | 除法赋值 |
| `%=` | `$a = $a % $b` | 取模赋值 |
| `**=` | `$a = $a ** $b` | 指数赋值 |
| `.=` | `$a = $a . $b` | 字符串连接赋值 |

### 自增自减

**前置**：`++$a`、`--$a`（先运算后使用）
**后置**：`$a++`、`$a--`（先使用后运算）

### 数组解构赋值

**语法**：`[$var1, $var2] = $array;`
**要求**：PHP 7.1+

## 基本用法

### 基本赋值示例

```php
<?php
declare(strict_types=1);

$a = 10;
$b = $a;
```

### 复合赋值示例

```php
<?php
declare(strict_types=1);

$a = 10;
$a += 5;  // $a = 15
$a -= 3;  // $a = 12
$a *= 2;  // $a = 24
```

### 自增自减示例

```php
<?php
declare(strict_types=1);

$a = 10;
$b = ++$a;  // $a = 11, $b = 11
$c = $a++;  // $a = 12, $c = 11
```

### 数组解构赋值示例

```php
<?php
declare(strict_types=1);

[$a, $b] = [1, 2];  // $a = 1, $b = 2
['name' => $name, 'age' => $age] = ['name' => 'John', 'age' => 25];
```

## 使用场景

- **变量赋值**：给变量赋值
- **计数器**：使用自增自减实现计数器
- **数组处理**：使用数组解构提取值

## 注意事项

- **优先级**：赋值运算符优先级较低
- **前置后置**：理解前置和后置的区别
- **数组解构**：确保数组结构匹配

## 常见问题

### 问题 1：赋值运算符优先级
- **原因**：赋值运算符优先级较低
- **解决**：使用括号明确优先级

### 问题 2：前置后置混淆
- **原因**：不理解前置和后置的区别
- **解决**：理解执行顺序

## 最佳实践

- 使用复合赋值简化代码
- 理解前置和后置的区别
- 使用数组解构提高代码可读性
- 注意赋值运算符的优先级
