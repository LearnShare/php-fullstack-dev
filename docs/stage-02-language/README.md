# 阶段二：PHP 基础语法与语言精通

本阶段覆盖 PHP 语言从语法、类型系统到错误处理的全栈知识，面向零基础学员设计。每章提供概念解释、示例代码、练习建议与常见错误对照，帮助你迅速建立坚实的后端编码能力。建议按顺序学习，每完成一章即编写 3~5 个小练习，并记录在学习日志中。

1. [`chapter-01-syntax`](chapter-01-syntax/README.md)：语法基础、文件结构与编码规范。
2. [`chapter-02-output`](chapter-02-output/README.md)：输出函数与调试技巧。
3. [`chapter-03-variables`](chapter-03-variables/README.md)：变量、常量与引用。
4. [`chapter-04-types`](chapter-04-types/README.md)：数据类型与检测。
5. [`chapter-05-type-casting`](chapter-05-type-casting/README.md)：类型转换与比较。
6. [`chapter-06-expressions`](chapter-06-expressions/README.md)：表达式与运算符。
7. [`chapter-07-strings`](chapter-07-strings/README.md)：字符串处理与多字节支持。
8. [`chapter-08-arrays`](chapter-08-arrays/README.md)：数组操作与高阶函数。
9. [`chapter-09-control-flow`](chapter-09-control-flow/README.md)：控制结构与循环。
10. [`chapter-10-functions`](chapter-10-functions/README.md)：函数、作用域与静态变量。
11. [`chapter-11-closures`](chapter-11-closures/README.md)：匿名函数、闭包与类型声明。
12. [`chapter-12-superglobals`](chapter-12-superglobals/README.md)：超级全局变量与安全处理。
13. [`chapter-13-modularity`](chapter-13-modularity/README.md)：文件引入与 Composer autoload。
14. [`chapter-14-filesystem`](chapter-14-filesystem/README.md)：文件系统与 I/O。
15. [`chapter-15-datetime`](chapter-15-datetime/README.md)：时间日期 API 与国际化。
16. [`chapter-16-null-system`](chapter-16-null-system/README.md)：isset、empty 与 Null 合并运算。
17. [`chapter-17-errors`](chapter-17-errors/README.md)：错误、异常与 Throwable。
18. [`chapter-18-standards`](chapter-18-standards/README.md)：代码规范、PHPDoc 与 PSR。
19. [`chapter-19-summary`](chapter-19-summary/README.md)：阶段总结与练习指引。
20. [`chapter-20-php-versions`](chapter-20-php-versions/README.md)：PHP 8.2-8.5 版本新特性。
21. [`chapter-21-debugging`](chapter-21-debugging/README.md)：常见错误与调试技巧。

完成阶段二后，你将具备：

- 扎实的 PHP 语言基础与调试能力，能够独立说明一段 PHP 程序的执行过程。
- 对类型系统、内置函数及运行机制的系统理解，能准确选择合适的数据结构。
- 具备规范意识：使用 PHPDoc、PSR-12，并借助静态分析工具自查。
- 形成“阅读 → 实验 → 笔记 → 练习 → 复盘”的学习闭环，提升代码阅读与重构速度。
