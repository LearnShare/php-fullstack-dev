# 2.11.3 使用导入的内容

## 概述

导入文件后，需要使用导入的内容。本节介绍如何使用导入的函数、类、配置、常量和变量，检查内容是否存在，处理名称冲突。

**章节类型**：实践性章节

**主要内容**：
- 使用导入的函数
- 使用导入的类
- 使用导入的配置和常量
- 使用导入的变量
- 检查内容是否存在
- 处理名称冲突
- 使用场景和最佳实践

## 核心内容

### 使用导入的函数

**直接调用**：导入后直接调用函数
**检查存在**：使用 `function_exists()` 检查

### 使用导入的类

**实例化**：使用 `new` 关键字实例化
**检查存在**：使用 `class_exists()` 检查

### 使用导入的配置

**直接使用**：导入后直接使用常量
**检查存在**：使用 `defined()` 检查

## 基本用法

### 使用函数示例

```php
<?php
declare(strict_types=1);

require 'math.php';

$result = add(3, 4);  // 使用导入的函数
```

### 使用类示例

```php
<?php
declare(strict_types=1);

require 'User.php';

$user = new User();  // 使用导入的类
```

### 检查存在示例

```php
<?php
declare(strict_types=1);

if (function_exists('add')) {
    $result = add(3, 4);
}

if (class_exists('User')) {
    $user = new User();
}
```

## 使用场景

- **代码复用**：使用导入的函数和类
- **配置管理**：使用导入的配置
- **动态加载**：根据条件加载和使用模块

## 注意事项

- **检查存在**：使用前检查内容是否存在
- **名称冲突**：注意避免名称冲突
- **作用域**：理解导入内容的作用域

## 常见问题

### 问题 1：函数未定义错误
- **原因**：函数未导入或不存在
- **解决**：检查导入和使用 `function_exists()`

### 问题 2：名称冲突
- **原因**：导入的内容与现有内容同名
- **解决**：使用命名空间或重命名

## 最佳实践

- 使用前检查内容是否存在
- 避免名称冲突
- 理解导入内容的作用域
- 使用有意义的名称
