# 2.19 本阶段总结与练习

## 目标回顾

完成阶段二后，你应具备以下能力：

1. 使用 PHP 的变量、类型、数组、控制结构、函数等核心语法构建完整脚本。
2. 理解类型系统、字符串/数组操作、错误处理、模块化组织与代码规范。
3. 能阅读、调试、扩展现有 PHP 项目，并为后续 OOP、架构、数据库章节打下基础。

## 综合练习建议

1. **CLI 工具**：编写一个命令行应用，实现待办事项（新增、列表、完成、删除），练习输入输出、文件存储、命令解析。
2. **表单处理**：构建一个多步骤表单（注册/登录），涵盖 `$_POST` 处理、Session、表单验证、错误反馈。
3. **配置驱动应用**：按照 `config/*.php` + autoload 方式组织代码，实现一个可扩展的“通知发送器”，支持 Email、SMS、Webhook 等不同驱动。
4. **日志与错误处理**：实现全局异常捕获，将异常记录到 `storage/logs/app.log`，并返回统一的 JSON/HTML 错误响应。
5. **练习题打卡**：每天完成 1-2 个 PHP 官网手册或 online judge 练习，巩固语法。

## 自我评估清单

- [ ] 可以清晰解释 `declare(strict_types=1)` 的作用。
- [ ] 知道 `isset`、`empty`、`??` 的差异并能正确使用。
- [ ] 能编写带类型声明、DocBlock、命名空间的函数/类。
- [ ] 会使用 Composer autoload 与 PSR-4 组织项目结构。
- [ ] 能使用 Xdebug/VarDumper 调试变量与函数执行流程。
- [ ] 能阅读 `phpinfo()` 或 `php -i` 输出，定位配置问题。

## 下一步

- 继续阶段三：面向对象、架构与设计模式。
- 尝试将 CLI 脚本迁移到 MVC 或微框架结构（Slim、Lumen）。
- 学习单元测试与静态分析工具，确保代码质量。

> 建议将阶段二的所有练习、笔记、代码整理到成熟的 Git 仓库，并记录遇到的问题与解决方案，为后续阶段复习参考。
