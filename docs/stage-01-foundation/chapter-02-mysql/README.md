# 1.2 MySQL 环境搭建与工具

## 目标

- 让零基础学员在一台电脑上成功运行 MySQL 8.0+，并明白每一步在做什么。
- 选择合适的 GUI/CLI 工具，建立安全连接、创建数据库与用户。
- 了解数据目录、字符集、备份策略，具备排查常见故障的能力。

## 章节内容

本章分为两个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[MySQL 安装与配置](section-01-installation.md)**：版本选择、安装方式（Docker/Windows/macOS/Linux）、配置（字符集、基本配置）、第一个数据库、连接安全、数据目录与备份及完整示例。

2. **[客户端工具与命令](section-02-tools-commands.md)**：客户端工具选择（命令行/图形化）、常用命令（`mysql`、`mysqldump`、`mysql_config_editor`、`mysqladmin`）、SQL 常用命令、常见问题排查及完整示例。

## 版本与发行版

- **MySQL Community 8.0**：默认选项，提供 InnoDB、GTID 复制
- **Percona Server / MariaDB**：仅在明确需求时使用
- 建议统一 `utf8mb4` 字符集、`InnoDB` 存储引擎

## 学习建议

1. **按顺序学习**：先学习安装和配置，再学习工具和命令。

2. **重点掌握**：
   - Docker 和本地安装方法
   - 数据库和用户的创建
   - 常用命令的使用
   - 问题排查方法

3. **实践练习**：
   - 完成每小节后的练习题目
   - 使用不同工具管理数据库
   - 实现备份和恢复流程

## 完成本章后

- 能够使用 Docker 和本地安装两种方式运行 MySQL。
- 能够创建数据库、用户、授权并记录在脚本中。
- 知道数据目录位置，能够执行逻辑备份与恢复。
- 面对常见报错（端口、权限、字符集）能查日志并修复。
