# 1.4 核心工具链

## 目标

- 掌握 Composer 依赖管理和自动加载。
- 配置 IDE 和代码格式化工具。

## 章节内容

本章分为两个独立小节，每节提供简洁的概念介绍、基本使用和完整示例：

1. **[1.4.1 Composer 基础](section-01-composer-basics.md)**：Composer 安装、基本使用、常用命令及完整示例。

2. **[1.4.2 IDE 与扩展配置](section-02-ide-extensions.md)**：IDE 选择（PhpStorm、VS Code）、扩展推荐、EditorConfig、代码格式化、静态分析工具及完整示例。

## 工具链概览

现代 PHP 开发工具链包括：

- **依赖管理**：Composer
- **代码编辑**：PhpStorm / VS Code

## 学习建议

1. **按顺序学习**：按顺序学习两个小节，理解每个工具的作用和配置方法。

2. **重点掌握**：
   - Composer 的基本使用和包管理
   - IDE 的配置和扩展安装

3. **实践练习**：
   - 完成每小节后的练习题目
   - 配置一个完整的开发环境

## 完成本章后

- 能够使用 Composer 管理项目依赖。
- 能够配置 IDE 和代码格式化工具。
