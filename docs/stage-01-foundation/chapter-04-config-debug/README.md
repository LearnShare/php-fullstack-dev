# 1.4 配置、扩展与调试

## 目标

- 掌握 php.ini 的配置方法和常用配置项。
- 了解如何安装和管理 PHP 扩展。
- 熟练配置和使用 Xdebug 进行调试。
- 掌握常用的调试技巧和工具。

## 章节内容

本章分为四个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[php.ini 配置](section-01-php-ini.md)**：定位配置文件、常用配置项、开发/生产环境配置、运行时配置（`ini_set()`、`ini_get()`）、环境相关配置及完整示例。

2. **[扩展管理](section-02-extensions.md)**：扩展类型、检查扩展、安装方法（Windows/macOS/Linux/Docker）、常用扩展、扩展配置及完整示例。

3. **[Xdebug 调试配置](section-03-xdebug.md)**：Xdebug 功能、安装方法、配置参数、IDE 配置（PhpStorm、VS Code）、断点调试流程、调试技巧、常见问题排查、性能分析、代码覆盖率及完整示例。

4. **[调试技巧与实践](section-04-debugging-tips.md)**：基础调试函数（`var_dump()`、`print_r()`、`var_export()`）、错误日志（`error_log()`）、堆栈跟踪（`debug_backtrace()`）、Xdebug 函数、调试工具类、性能调试、配置差异追踪及完整示例。

## 配置概览

- **php.ini**：PHP 主配置文件
- **扩展**：提供额外功能的模块
- **Xdebug**：调试和性能分析工具
- **调试函数**：内置的调试工具

## 学习建议

1. **按顺序学习**：按顺序学习四个小节，理解配置、扩展和调试的完整流程。

2. **重点掌握**：
   - php.ini 的常用配置项
   - 扩展的安装和配置方法
   - Xdebug 的配置和使用
   - 常用的调试技巧

3. **实践练习**：
   - 完成每小节后的练习题目
   - 配置开发和生产环境
   - 使用 Xdebug 进行实际调试

## 完成本章后

- 能够配置 php.ini 文件，适应不同环境需求。
- 能够安装和管理 PHP 扩展。
- 能够配置和使用 Xdebug 进行调试。
- 能够使用各种调试工具和技巧解决问题。
