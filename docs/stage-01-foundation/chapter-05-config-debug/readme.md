# 1.5 配置、扩展与调试基础

## 目标

- 掌握 php.ini 的配置方法和常用配置项。
- 了解如何安装和管理 PHP 扩展。
- 了解调试基础概念和工具。

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[1.5.1 php.ini 配置](section-01-php-ini.md)**：定位配置文件、常用配置项、开发/生产环境配置、运行时配置（`ini_set()`、`ini_get()`）、环境相关配置及完整示例。

2. **[1.5.2 扩展管理](section-02-extensions.md)**：扩展类型、检查扩展、安装方法（Windows/macOS/Linux/Docker）、常用扩展、扩展配置及完整示例。

3. **[1.5.3 调试基础介绍](section-03-debugging-basics.md)**：调试基础概念、基础调试函数（`var_dump()`、`print_r()`、`var_export()`）、错误日志（`error_log()`）、堆栈跟踪（`debug_backtrace()`）、调试工具介绍及完整示例。

## 配置概览

- **php.ini**：PHP 主配置文件
- **扩展**：提供额外功能的模块
- **调试工具**：帮助排查问题的工具

## 学习建议

1. **按顺序学习**：按顺序学习三个小节，理解配置、扩展和调试的完整流程。

2. **重点掌握**：
   - php.ini 的常用配置项
   - 扩展的安装和配置方法
   - 基础调试工具的使用

3. **实践练习**：
   - 完成每小节后的练习题目
   - 配置开发和生产环境
   - 使用调试工具进行实际调试

## 完成本章后

- 能够配置 php.ini 文件，适应不同环境需求。
- 能够安装和管理 PHP 扩展。
- 能够使用基础调试工具解决问题。
