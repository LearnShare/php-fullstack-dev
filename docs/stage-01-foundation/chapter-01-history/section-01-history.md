# 1.1.1 PHP 起源与发展历程

## 概述

PHP（PHP: Hypertext Preprocessor）是一种广泛使用的开源服务器端脚本语言，特别适合 Web 开发。本节介绍 PHP 的起源、发展历程和重要里程碑。

**章节类型**：概念性章节

**主要内容**：
- PHP 的诞生背景和初衷
- 各主要版本的演进历程
- 重要里程碑和特性发展
- 性能演进对比
- 社区发展情况
- 完整时间线总结

## 核心内容

### PHP 的诞生（1994-1995）
- Rasmus Lerdorf 的初衷
- PHP/FI（1995）第一个公开版本
- 早期功能特点

### PHP 3.0（1998）- 真正的 PHP
- 重要里程碑
- 名称变更：从 "Personal Home Page" 到 "PHP: Hypertext Preprocessor"
- 语法改进和扩展性
- 主要特性

### PHP 4.0（2000）- Zend Engine
- Zend Engine 的引入
- 性能提升
- 会话管理和输出缓冲
- 对 PHP 发展的影响

### PHP 5.0（2004）- 面向对象革命
- 完整的 OOP 支持
- 异常处理机制
- PDO 数据库接口
- Zend Engine 2
- 代码示例

### PHP 5.x 系列（2005-2012）
- PHP 5.1 - 5.6 各版本特性
- 命名空间（5.3）
- Traits（5.4）
- 生成器（5.5）
- 可变参数（5.6）

### PHP 7.0（2015）- 性能革命
- 性能提升（2-3倍）
- 标量类型声明
- 返回类型声明
- 空合并运算符
- 太空船运算符
- Zend Engine 3
- 性能对比表格

### PHP 7.x 系列（2016-2020）
- PHP 7.1 - 7.4 各版本特性
- 可空类型（7.1）
- 类型属性（7.4）
- 箭头函数（7.4）

### PHP 8.0（2020）- 现代 PHP
- JIT 编译
- 命名参数
- 联合类型
- match 表达式
- 构造器属性提升
- Attributes（注解）

### PHP 8.1（2021）
- 枚举（Enums）
- 只读属性
- first-class callable
- 交集类型

### PHP 8.2+（2022-2025）
- PHP 8.2 新特性（Readonly 类、独立类型等）
- PHP 8.3 新特性（json_validate、类常量类型等）
- PHP 8.4 新特性（属性钩子、不对称可见性等）
- PHP 8.5 新特性（管道操作符、clone with 等）
- 版本选择建议
- 版本支持周期表格

- 重要组织（PHP Group、Zend、PHP-FIG、Composer）

- 完整版本发布时间线表格

## 总结

- PHP 从简单的脚本语言发展为现代编程语言
- 性能提升显著，特别是 PHP 7.0 和 PHP 8.0
- 类型系统和 OOP 功能不断完善
- 社区活跃，生态系统丰富
- 建议使用仍在安全支持期内的版本（PHP 8.2+）
