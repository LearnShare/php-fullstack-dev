# 1.1.1 PHP 起源与发展历程

## 概述

PHP（PHP: Hypertext Preprocessor）是一种广泛使用的开源服务器端脚本语言，特别适合 Web 开发。本节介绍 PHP 的 Hello World 示例、发展历程、重要里程碑和应用场景。

## Hello World

### 第一个 PHP 程序

创建一个名为 `hello.php` 的文件，内容如下：

```php
<?php
echo "Hello, World!";
```

**运行方式**：

1. **命令行运行**：
   ```bash
   php hello.php
   ```
   输出：`Hello, World!`

2. **Web 服务器运行**：
   - 将文件放在 Web 服务器目录中
   - 通过浏览器访问：`http://localhost/hello.php`
   - 浏览器显示：`Hello, World!`

### 代码说明

- `<?php`：PHP 代码开始标记
- `echo`：输出语句，用于向浏览器或命令行输出内容
- `"Hello, World!"`：要输出的字符串
- **注意**：纯 PHP 文件（只包含 PHP 代码，没有 HTML）不应该使用结束标签 `?>`，这是 PSR-1 标准和现代 PHP 最佳实践，可以避免意外的空白字符输出

## 发展历史

### PHP 的诞生（1994-1995）

- **1994**：Rasmus Lerdorf 创建了 PHP，最初用于管理个人主页
- **1995**：PHP/FI（Personal Home Page Tools/Form Interpreter）第一个公开版本发布

### PHP 3.0（1998）- 真正的 PHP

- 重要里程碑：PHP 3.0 标志着 PHP 成为真正的编程语言
- 名称变更：从 "Personal Home Page" 到 "PHP: Hypertext Preprocessor"（递归缩写）
- 语法改进：支持更多数据类型和函数
- 扩展性：支持扩展模块

### PHP 4.0（2000）- Zend Engine

- **Zend Engine**：引入新的执行引擎，提升性能
- **会话管理**：内置会话支持
- **输出缓冲**：支持输出缓冲功能

### PHP 5.0（2004）- 面向对象革命

- **完整的 OOP 支持**：类、对象、继承、接口、抽象类
- **异常处理**：try-catch 异常处理机制
- **PDO**：统一的数据库访问接口
- **Zend Engine 2**：新的执行引擎

### PHP 7.0（2015）- 性能革命

- **性能提升**：相比 PHP 5.6 性能提升 2-3 倍
- **类型声明**：支持标量类型声明和返回类型声明
- **新运算符**：空合并运算符（`??`）、太空船运算符（`<=>`）
- **Zend Engine 3**：全新的执行引擎

### PHP 8.0（2020）- 现代 PHP

- **JIT 编译**：即时编译，进一步提升性能
- **命名参数**：支持命名参数调用
- **联合类型**：支持多种类型联合
- **match 表达式**：更强大的 switch 替代
- **构造器属性提升**：简化类属性定义
- **Attributes（注解）**：支持注解语法

### PHP 8.1+（2021-2025）

- **PHP 8.1**：枚举（Enums）、只读属性
- **PHP 8.2**：Readonly 类、独立类型
- **PHP 8.3**：json_validate、类常量类型
- **PHP 8.4**：属性钩子、不对称可见性
- **PHP 8.5**：管道操作符、clone with

### 版本选择建议

- **开发环境**：建议使用最新稳定版本（PHP 8.3+）
- **生产环境**：建议使用仍在安全支持期内的版本（PHP 8.2+）
- **版本支持周期**：每个主要版本支持约 3 年

## 应用场景

### Web 开发

PHP 最广泛的应用场景是 Web 开发：

- **动态网站**：用户交互、表单处理、内容管理
- **Web 应用**：博客、论坛、电商、社交网络
- **API 服务**：RESTful API、GraphQL API
- **内容管理系统（CMS）**：WordPress、Drupal、Joomla

### 命令行工具

PHP 也可以用于开发命令行工具：

- **系统管理脚本**：自动化任务、系统维护
- **数据处理工具**：数据导入导出、批量处理
- **开发工具**：代码生成、项目脚手架

### 企业应用

PHP 在企业级应用中也广泛应用：

- **企业网站**：公司官网、产品展示
- **内部系统**：OA 系统、CRM 系统、ERP 系统
- **微服务架构**：作为微服务的一部分

### 学习与教育

PHP 也是学习编程的优秀选择：

- **语法简单**：易于学习和理解
- **文档丰富**：官方文档和社区资源丰富
- **生态完善**：框架和工具齐全

## 总结

- PHP 从简单的脚本语言发展为现代编程语言
- 性能提升显著，特别是 PHP 7.0 和 PHP 8.0
- 类型系统和 OOP 功能不断完善
- 主要应用于 Web 开发，也可用于命令行工具和企业应用
- 建议使用仍在安全支持期内的版本（PHP 8.2+）
