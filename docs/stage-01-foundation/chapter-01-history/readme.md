# 1.1 PHP 发展史

## 目标

- 了解 PHP 的基本概念和 Hello World 示例。
- 理解 PHP 的发展历程和重要里程碑。
- 了解 PHP 的主要应用场景。
- 了解 PHP 生态系统中主流的工具、库和框架。

## 章节内容

本章分为两个独立小节，每节提供简洁的概念介绍和核心信息：

1. **[1.1.1 PHP 起源与发展历程](section-01-history.md)**：PHP 的 Hello World 示例、发展历程、重要里程碑和应用场景。

2. **[1.1.2 PHP 生态系统](section-02-ecosystem.md)**：PHP 生态系统中主流的框架、工具、库和标准规范。

## 核心概念

- **PHP**：PHP: Hypertext Preprocessor（递归缩写）
- **Hello World**：第一个 PHP 程序示例
- **版本演进**：从脚本语言到现代编程语言的转变
- **应用场景**：Web 开发、命令行工具、企业应用
- **生态系统**：框架、工具、库和标准规范

## 学习建议

1. **按顺序学习**：按顺序学习两个小节，理解 PHP 的基本概念和生态系统。

2. **重点掌握**：
   - PHP 的 Hello World 示例和基本语法
   - PHP 的发展历程和重要里程碑
   - PHP 的主要应用场景
   - PHP 生态系统中主流的框架和工具

3. **实践练习**：
   - 编写并运行第一个 PHP 程序（Hello World）
   - 了解当前使用的 PHP 版本
   - 查看 PHP 官方文档了解最新特性
   - 探索 PHP 生态系统中的框架和工具

## 完成本章后

- 能够编写并运行第一个 PHP 程序（Hello World）。
- 能够理解 PHP 的发展历程和重要里程碑。
- 了解 PHP 的主要应用场景。
- 了解 PHP 生态系统中主流的框架、工具、库和标准规范。
- 能够根据项目需求选择合适的 PHP 版本和工具。
