# 1.1 PHP 安装与运行基础

## 目标

- 了解 PHP 在不同平台的安装方式与命令行工具，初学者也能独立完成部署。
- 选择合适的 PHP 8.2+ 版本，并掌握并行维护 CLI、FPM、容器运行时的方法。
- 通过验证命令、示例脚本与排错指南，建立"安装→验证→记录"的闭环流程。

## 章节内容

本章分为两个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[PHP 安装与版本管理](section-01-installation.md)**：版本策略、安装方式（Windows/macOS/Linux/Docker）、多版本管理、运行验证、常用命令、常见故障及完整示例。

2. **[运行模式与验证](section-02-execution-modes.md)**：运行模式对比、CLI 模式、内置 Web 服务器、PHP-FPM 模式、常驻运行时、运行验证及完整示例。

## 版本策略

| 场景 | 推荐版本 | 说明 |
| :--- | :------- | :--- |
| 本地开发 | 8.2 LTS | 与主流框架兼容，扩展生态成熟 |
| 新项目 | 8.3 | 获得最新语法特性 |
| 旧项目维护 | 8.1 | 如依赖旧扩展，可暂时保留 |

## 运行模式

- **CLI**：用于执行脚本、迁移、队列；命令为 `php script.php`
- **PHP-FPM**：与 Nginx/Apache 配合的常规 Web 模式
- **内置服务器**：`php -S 127.0.0.1:8000`，适合教学和 Demo
- **常驻运行时**：FrankenPHP、RoadRunner、Swoole 通过常驻内存提升性能

## 学习建议

1. **按顺序学习**：先学习安装和版本管理，再学习运行模式。

2. **重点掌握**：
   - 不同平台的安装方法
   - 多版本管理技巧
   - 不同运行模式的特点
   - 验证和故障排查方法

3. **实践练习**：
   - 完成每小节后的练习题目
   - 在本地环境安装 PHP
   - 验证不同运行模式

## 完成本章后

- 能够在任意操作系统上安装 PHP。
- 能够管理多个 PHP 版本。
- 能够理解不同运行模式的差异。
- 能够验证 PHP 安装和配置。
