# 1.2 PHP 安装与运行基础

## 目标

- 了解 PHP 在不同平台的安装方式。
- 掌握 PHP 版本管理方法。
- 理解 PHP 的不同运行模式。
- 能够验证 PHP 安装和配置。

## 章节内容

本章分为两个独立小节，每节提供简洁的概念介绍、安装步骤和完整示例：

1. **[1.2.1 PHP 安装与版本管理](section-01-installation.md)**：不同平台的安装方法（Windows/macOS/Linux/Docker）、版本管理、运行验证、常用命令和常见问题。

2. **[1.2.2 运行模式与验证](section-02-execution-modes.md)**：运行模式对比（CLI、内置服务器、PHP-FPM、常驻运行时）、使用方法和验证方式。

## 版本策略

| 场景 | 推荐版本 | 说明 |
|:-----|:---------|:-----|
| 本地开发 | 8.2+ | 与主流框架兼容，扩展生态成熟 |
| 新项目 | 8.3+ | 获得最新语法特性 |
| 旧项目维护 | 8.1+ | 如依赖旧扩展，可暂时保留 |

## 运行模式

- **CLI**：用于执行脚本、迁移、队列；命令为 `php script.php`
- **内置服务器**：`php -S localhost:8000`，适合教学和 Demo
- **PHP-FPM**：与 Nginx/Apache 配合的常规 Web 模式
- **常驻运行时**：RoadRunner、FrankenPHP、OpenSwoole 通过常驻内存提升性能

## 学习建议

1. **按顺序学习**：先学习安装和版本管理，再学习运行模式。

2. **重点掌握**：
   - 不同平台的安装方法
   - 版本验证和常用命令
   - 不同运行模式的特点和使用方法
   - 运行模式验证方法

3. **实践练习**：
   - 在本地环境安装 PHP
   - 验证 PHP 安装和配置
   - 尝试不同的运行模式
   - 完成每小节后的练习题目

## 完成本章后

- 能够在任意操作系统上安装 PHP。
- 能够验证 PHP 安装和配置。
- 能够理解不同运行模式的差异。
- 能够根据场景选择合适的运行模式。
