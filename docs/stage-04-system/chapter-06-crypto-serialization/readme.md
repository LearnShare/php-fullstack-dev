# 4.6 加密、哈希与序列化

## 目标

- 掌握 PHP 中密码哈希、数据加密、哈希函数的使用方法。
- 理解序列化与反序列化的概念和应用场景。
- 掌握数据持久化的安全方法，能够安全地存储和传输敏感数据。

## 章节内容

本章分为五个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[4.6.1 密码哈希](section-01-password-hashing.md)**：密码哈希概述、password_hash()、password_verify()、哈希算法选择、完整示例。

2. **[4.6.2 数据加密](section-02-data-encryption.md)**：数据加密概述、openssl_encrypt()、openssl_decrypt()、加密算法选择、密钥管理、完整示例。

3. **[4.6.3 哈希函数](section-03-hash-functions.md)**：哈希函数概述、hash()、hash_hmac()、常见哈希算法、完整性验证、完整示例。

4. **[4.6.4 序列化与反序列化](section-04-serialization.md)**：序列化概念、serialize()、unserialize()、JSON 序列化、安全注意事项、完整示例。

5. **[4.6.5 数据持久化](section-05-data-persistence.md)**：数据持久化概述、安全存储策略、敏感数据处理、完整示例。

## 核心概念

- **密码哈希**：单向加密，用于密码存储
- **数据加密**：双向加密，用于数据保护
- **哈希函数**：数据完整性验证
- **序列化**：对象转换为可存储格式
- **反序列化**：从存储格式恢复对象

## 学习建议

1. **按顺序学习**：按顺序学习五个小节，理解加密、哈希与序列化的完整体系。

2. **重点掌握**：
   - 密码哈希的正确使用
   - 数据加密方法
   - 哈希函数应用
   - 序列化安全

3. **实践练习**：
   - 完成每小节后的练习题目
   - 实现安全的密码存储
   - 理解加密和哈希的区别

## 完成本章后

- 能够安全地存储密码。
- 能够加密和解密数据。
- 能够使用哈希函数验证数据完整性。
- 能够安全地进行序列化和反序列化。
