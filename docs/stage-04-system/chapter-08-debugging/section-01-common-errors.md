## 基本用法

### 错误排查示例

```php
<?php
declare(strict_types=1);

// 启用错误报告
error_reporting(E_ALL);
ini_set('display_errors', '1');

// 使用 try-catch 捕获异常
try {
    // 代码
} catch (Exception $e) {
    // 查看错误信息
    echo $e->getMessage();
    echo $e->getTraceAsString();
}
```
