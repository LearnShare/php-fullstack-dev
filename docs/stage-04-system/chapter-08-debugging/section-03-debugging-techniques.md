## 基本用法

### 调试技巧示例

```php
<?php
declare(strict_types=1);

// 使用 var_dump 调试
var_dump($variable);

// 使用 debug_backtrace 追踪调用
$trace = debug_backtrace();
print_r($trace);

// 日志调试
error_log('调试信息: ' . print_r($data, true));
```
