# 4.3 网络编程基础

## 目标

- 理解网络编程的基本概念，掌握 Socket 编程的基础知识。
- 掌握 HTTP 客户端编程方法，能够使用 PHP 进行网络请求。
- 理解网络协议的基本原理，了解 TCP/IP、HTTP 等协议的工作机制。

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[4.3.1 Socket 编程基础](section-01-socket-basics.md)**：Socket 概念、TCP Socket、UDP Socket、服务器端编程、客户端编程、完整示例。

2. **[4.3.2 HTTP 客户端编程](section-02-http-client.md)**：HTTP 请求方法、file_get_contents()、cURL 扩展、Guzzle 库、请求头设置、响应处理、完整示例。

3. **[4.3.3 网络协议理解](section-03-network-protocols.md)**：TCP/IP 协议栈、HTTP 协议、HTTPS 协议、WebSocket 协议、协议选择、完整示例。

## 核心概念

- **Socket**：网络通信的端点
- **TCP**：可靠的传输协议
- **UDP**：不可靠但快速的传输协议
- **HTTP**：应用层协议
- **cURL**：强大的网络请求库

## 学习建议

1. **按顺序学习**：按顺序学习三个小节，理解网络编程的完整体系。

2. **重点掌握**：
   - Socket 编程基础
   - HTTP 客户端编程
   - 网络协议的基本理解

3. **实践练习**：
   - 完成每小节后的练习题目
   - 实现简单的网络客户端
   - 理解网络通信的基本流程

## 完成本章后

- 能够进行基础的 Socket 编程。
- 能够使用 PHP 进行 HTTP 请求。
- 能够理解网络协议的基本原理。
- 能够选择合适的网络协议和工具。
