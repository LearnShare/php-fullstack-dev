# 4.1 时间与日期处理

## 目标

- 掌握 PHP 中 DateTime 类的使用，理解时间与日期处理的核心概念。
- 理解时区的概念，能够正确处理不同时区的时间转换。
- 掌握时间格式化与解析方法，能够在不同格式间转换。
- 掌握时间计算与比较方法，能够进行时间的加减运算和比较操作。

## 章节内容

本章分为四个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[4.1.1 DateTime 基础](section-01-datetime-basics.md)**：DateTime 类的基本用法、创建 DateTime 对象、获取当前时间、时间格式化基础。

3. **[4.1.3 格式化与解析](section-03-formatting-parsing.md)**：日期格式化、日期解析、格式字符串详解、国际化处理。

4. **[4.1.4 时间计算与比较](section-04-calculations-comparisons.md)**：时间加减运算、时间间隔（DateInterval）、时间比较、日期差计算。

## 核心概念

- **DateTime**：PHP 中处理日期和时间的主要类
- **时区（Timezone）**：全球不同地区的时间标准
- **格式化（Format）**：将日期时间转换为字符串格式
- **解析（Parse）**：将字符串转换为 DateTime 对象
- **时间间隔（DateInterval）**：表示时间差值的对象
