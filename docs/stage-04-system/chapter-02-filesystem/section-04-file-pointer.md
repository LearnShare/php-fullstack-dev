# 4.2.4 文件指针操作

## 概述

文件指针是文件操作中的重要概念，用于标识文件中的当前位置。本节介绍 PHP 中文件指针的操作方法，包括 fseek()、ftell()、rewind() 等函数，帮助零基础学员理解文件指针的概念和使用。

## 基本用法

### 文件指针操作示例

```php
<?php
declare(strict_types=1);

$handle = fopen('data.txt', 'r');

// 移动到文件末尾
fseek($handle, 0, SEEK_END);

// 获取当前位置
$position = ftell($handle);

// 重置到开头
rewind($handle);
```
