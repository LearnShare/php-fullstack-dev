## 常见问题

### Q: 如何发布 Composer 包到 Packagist？

A: 在 Packagist 注册账号，提交 GitHub 仓库链接，Packagist 会自动同步。

### Q: 包的版本如何管理？

A: 使用语义化版本（Semantic Versioning），通过 Git 标签管理版本。
