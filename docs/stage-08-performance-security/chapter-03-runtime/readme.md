# 8.3 现代 PHP 运行时

## 目标

- 理解现代 PHP 运行时的概念和优势。
- 掌握 RoadRunner、FrankenPHP、OpenSwoole、Laravel Octane 等现代运行时。

## 核心概念

- **现代 PHP 运行时**：高性能的 PHP 应用运行时环境
- **RoadRunner**：Go 语言编写的高性能 PHP 应用服务器
- **FrankenPHP**：基于 Caddy 的现代 PHP 运行时
- **OpenSwoole**：高性能 PHP 协程框架
- **Laravel Octane**：Laravel 应用的高性能运行时

## 学习建议

1. **按顺序学习**：按顺序学习五个小节，理解现代 PHP 运行时的完整体系。

2. **重点掌握**：
   - 各运行时的特点和优势
   - 安装和配置方法
   - 应用部署方式
   - 性能对比和选型

3. **实践练习**：
   - 完成每小节后的练习题目
   - 部署应用到不同运行时
   - 对比性能表现

## 完成本章后

- 能够理解现代 PHP 运行时的概念和优势。
- 能够安装和配置 RoadRunner、FrankenPHP、OpenSwoole、Laravel Octane。
- 能够将应用部署到现代运行时。
- 能够根据需求选择合适的运行时。
