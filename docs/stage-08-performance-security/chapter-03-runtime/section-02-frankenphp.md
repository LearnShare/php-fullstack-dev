## 常见问题

### Q: FrankenPHP 适用于哪些应用？

A: 适用于大多数 PHP 应用，特别是需要高性能的 Web 应用和 API。

### Q: 如何配置进程数？

A: 根据服务器资源和应用负载配置，建议从 4-8 个进程开始。
