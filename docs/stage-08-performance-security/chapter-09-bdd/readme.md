# 8.9 BDD（行为驱动开发）

## 目标

- 理解 BDD 的概念和工作流程。
- 掌握 Behat、Codeception 等 BDD 工具的使用方法。
- 能够在实际开发中应用 BDD。

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[8.9.1 BDD 概述](section-01-overview.md)**：BDD 概述、BDD 概念、BDD 优势、BDD 原则、完整示例。

2. **[8.9.2 BDD 工具（Behat、Codeception）](section-02-tools.md)**：BDD 工具概述、Behat 使用、Codeception 使用、工具对比、完整示例。

3. **[8.9.3 BDD 实践](section-03-practice.md)**：BDD 实践案例、常见问题、最佳实践、完整示例。

## 核心概念

- **BDD**：行为驱动开发
- **Gherkin**：BDD 的语法格式
- **Behat**：PHP 的 BDD 框架
- **Codeception**：PHP 的测试框架

## 学习建议

1. **按顺序学习**：按顺序学习三个小节，理解 BDD 的完整体系。

2. **重点掌握**：
   - BDD 的概念和工作流程
   - Gherkin 语法
   - Behat 和 Codeception 的使用

3. **实践练习**：
   - 完成每小节后的练习题目
   - 使用 BDD 开发功能
   - 体验 BDD 的优势

## 完成本章后

- 能够理解 BDD 的概念和工作流程。
- 能够使用 Behat、Codeception 等 BDD 工具。
- 能够在实际开发中应用 BDD。
