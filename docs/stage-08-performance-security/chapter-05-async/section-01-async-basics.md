## 常见问题

### Q: 异步处理适用于哪些场景？

A: 适用于 I/O 密集型任务，如网络请求、文件操作、数据库查询等。

### Q: 如何避免回调地狱？

A: 使用 Promise 或 async/await 语法，使异步代码更易读。
