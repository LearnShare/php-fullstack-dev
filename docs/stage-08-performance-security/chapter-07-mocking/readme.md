- 理解 Mock、Stub、Fakes 的概念和区别。
- 掌握测试替身的使用方法。

## 核心概念

- **Mock**：模拟对象，验证交互
- **Stub**：存根对象，提供返回值
- **Fakes**：假对象，简化实现
- **测试替身**：测试中替代真实对象的对象

## 学习建议
