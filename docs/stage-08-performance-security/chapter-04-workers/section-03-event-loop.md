## 常见问题

### Q: Event Loop 如何工作？

A: Event Loop 持续监听事件队列，当有事件时处理，没有事件时等待，实现非阻塞 I/O。

### Q: 如何避免阻塞 Event Loop？

A: 使用异步 I/O 操作，避免同步阻塞操作，将耗时操作放到 Worker 进程。
