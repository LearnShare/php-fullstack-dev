# 8.1 安全防护

## 目标

- 理解 Web 应用安全的基本概念和常见威胁。
- 掌握 OWASP Top 10（2025）风险模型与防御方法。
- 能够实现密码加密、Secrets 管理、安全 HTTP 头等安全机制。
- 能够建立 API 安全、安全审计与日志、依赖安全等安全体系。

## 核心概念

- **OWASP Top 10**：Web 应用十大安全风险
- **密码哈希**：使用安全的哈希算法存储密码
- **加密**：使用加密算法保护敏感数据
- **Secrets 管理**：安全地管理密钥和敏感信息
- **安全 HTTP 头**：通过 HTTP 头增强安全性
- **API 安全**：保护 API 接口的安全机制
- **安全审计**：记录和监控安全事件
- **依赖安全**：确保第三方依赖的安全性

## 学习建议

1. **按顺序学习**：按顺序学习七个小节，理解安全防护的完整体系。

2. **重点掌握**：
   - OWASP Top 10 风险模型
   - 密码加密和存储
   - Secrets 管理方法
   - API 安全机制
   - 安全审计和日志

3. **实践练习**：
   - 完成每小节后的练习题目
   - 实现安全防护机制
   - 建立安全审计体系

## 完成本章后

- 能够配置安全 HTTP 头。
