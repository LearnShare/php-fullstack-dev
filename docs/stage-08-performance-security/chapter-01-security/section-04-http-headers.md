## 常见问题

### Q: 如何测试安全头配置？

A: 使用浏览器开发者工具或在线工具检查 HTTP 响应头。

### Q: 安全头配置会影响应用功能吗？

A: 某些安全头（如 CSP）可能影响应用功能，需要根据实际情况调整。
