## 常见问题

### Q: 如何发现依赖漏洞？

A: 使用漏洞扫描工具（如 Composer audit、Snyk）定期扫描依赖。

### Q: 如何处理依赖漏洞？

A: 优先更新到安全版本，如果无法更新，考虑替换或移除依赖。
