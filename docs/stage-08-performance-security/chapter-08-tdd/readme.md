# 8.8 TDD（测试驱动开发）

## 目标

- 理解 TDD 的概念和工作流程。
- 掌握 TDD 的开发方法。
- 能够在实际开发中应用 TDD。

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[8.8.1 TDD 概述](section-01-overview.md)**：TDD 概述、TDD 概念、TDD 优势、TDD 原则、完整示例。

2. **[8.8.2 TDD 工作流程](section-02-workflow.md)**：TDD 工作流程概述、Red-Green-Refactor 循环、实践步骤、完整示例。

3. **[8.8.3 TDD 实践](section-03-practice.md)**：TDD 实践案例、常见问题、最佳实践、完整示例。

## 核心概念

- **TDD**：测试驱动开发
- **Red-Green-Refactor**：TDD 的三个阶段
- **测试先行**：先写测试，再写实现

## 学习建议

1. **按顺序学习**：按顺序学习三个小节，理解 TDD 的完整体系。

2. **重点掌握**：
   - TDD 的概念和工作流程
   - Red-Green-Refactor 循环
   - TDD 的实践方法

3. **实践练习**：
   - 完成每小节后的练习题目
   - 使用 TDD 开发功能
   - 体验 TDD 的优势

## 完成本章后

- 能够理解 TDD 的概念和工作流程。
- 能够在实际开发中应用 TDD。
- 能够体验 TDD 带来的优势。
