# 10.10 基础设施即代码（IaC）

## 目标

- 理解基础设施即代码的概念和优势。
- 掌握 Terraform 和 Pulumi 的使用方法。
- 能够使用 IaC 工具管理基础设施。

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[10.10.1 基础设施即代码概述](section-01-overview.md)**：IaC 概述、IaC 优势、IaC 工具、完整示例。

2. **[10.10.2 Terraform](section-02-terraform.md)**：Terraform 概述、配置语法、资源管理、完整示例。

3. **[10.10.3 Pulumi](section-03-pulumi.md)**：Pulumi 概述、配置语法、资源管理、完整示例。

## 核心概念

- **基础设施即代码**：使用代码定义和管理基础设施
- **Terraform**：HashiCorp 的 IaC 工具
- **Pulumi**：支持多语言的 IaC 工具

## 学习建议

1. **按顺序学习**：按顺序学习三个小节，理解 IaC 的完整体系。

2. **重点掌握**：
   - IaC 概念和优势
   - Terraform 使用
   - Pulumi 使用

3. **实践练习**：
   - 完成每小节后的练习题目
   - 使用 Terraform 管理基础设施
   - 使用 Pulumi 管理基础设施

## 完成本章后

- 能够理解基础设施即代码的概念和优势。
- 能够使用 Terraform 和 Pulumi 管理基础设施。
- 能够应用 IaC 最佳实践。
