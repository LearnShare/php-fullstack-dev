# 10.2 docker-compose 与本地开发环境

## 目标

- 掌握 docker-compose 的使用方法。
- 能够配置本地开发环境。
- 理解多容器应用的编排。

## 章节内容

本章分为两个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[10.2.1 docker-compose 基础](section-01-compose-basics.md)**：docker-compose 概述、配置文件、常用命令、服务编排、完整示例。

2. **[10.2.2 本地开发环境](section-02-dev-environment.md)**：开发环境概述、环境配置、服务配置、开发工作流、完整示例。

## 核心概念

- **docker-compose**：用于定义和运行多容器 Docker 应用的工具
- **服务编排**：管理多个容器的启动和协作
- **开发环境**：本地开发使用的容器化环境

## 学习建议

1. **按顺序学习**：按顺序学习两个小节，理解 docker-compose 的完整体系。

2. **重点掌握**：
   - docker-compose 配置
   - 服务编排
   - 开发环境配置

3. **实践练习**：
   - 完成每小节后的练习题目
   - 配置开发环境
   - 使用 docker-compose 管理应用

## 完成本章后

- 能够使用 docker-compose 管理多容器应用。
- 能够配置本地开发环境。
- 能够理解服务编排的原理。
