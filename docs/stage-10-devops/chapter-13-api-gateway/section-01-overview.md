## 常见问题

### Q: API 网关和负载均衡有什么区别？

A: API 网关提供更丰富的功能（如认证、限流、监控），负载均衡主要提供流量分发。

### Q: 什么时候需要使用 API 网关？

A: 当有多个微服务、需要统一管理 API、需要统一认证授权时使用。
