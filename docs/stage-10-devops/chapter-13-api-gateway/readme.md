# 10.13 API 网关

## 目标

- 理解 API 网关的作用和优势。
- 掌握 Kong 和 Traefik 的使用方法。
- 能够使用 API 网关管理 API 服务。

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[10.13.1 API 网关概述](section-01-overview.md)**：API 网关概述、网关功能、网关优势、完整示例。

2. **[10.13.2 Kong](section-02-kong.md)**：Kong 概述、Kong 安装、Kong 配置、完整示例。

3. **[10.13.3 Traefik](section-03-traefik.md)**：Traefik 概述、Traefik 安装、Traefik 配置、完整示例。

## 核心概念

- **API 网关**：统一管理 API 服务的入口
- **Kong**：开源的 API 网关
- **Traefik**：现代的 HTTP 反向代理和负载均衡器

## 学习建议

1. **按顺序学习**：按顺序学习三个小节，理解 API 网关的完整体系。

2. **重点掌握**：
   - API 网关功能
   - Kong 使用
   - Traefik 使用

3. **实践练习**：
   - 完成每小节后的练习题目
   - 配置 API 网关
   - 管理 API 服务
