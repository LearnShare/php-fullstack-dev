## 常见问题

### Q: Kubernetes 和 Docker 有什么关系？

A: Kubernetes 是容器编排平台，可以管理 Docker 容器，提供更高级的容器管理功能。

### Q: 如何学习 Kubernetes？

A: 从核心概念开始，使用 minikube 或 kind 搭建本地环境，逐步学习。
