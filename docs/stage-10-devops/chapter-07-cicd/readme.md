# 10.7 CI/CD 与 GitHub Actions

## 目标

- 理解 CI/CD 的概念和工作流程。
- 掌握 GitHub Actions 和 GitLab CI 的使用方法。
- 能够实现自动化构建、测试和部署。

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[10.7.1 CI/CD 基础](section-01-cicd-basics.md)**：CI/CD 概述、CI/CD 流程、CI/CD 工具、完整示例。

2. **[10.7.2 GitHub Actions](section-02-github-actions.md)**：GitHub Actions 概述、工作流配置、常用操作、完整示例。

3. **[10.7.3 GitLab CI](section-03-gitlab-ci.md)**：GitLab CI 概述、CI/CD 配置、Pipeline 配置、完整示例。

## 核心概念

- **CI/CD**：持续集成和持续部署
- **GitHub Actions**：GitHub 的 CI/CD 平台
- **GitLab CI**：GitLab 的 CI/CD 平台

## 学习建议

1. **按顺序学习**：按顺序学习三个小节，理解 CI/CD 的完整体系。

2. **重点掌握**：
   - CI/CD 流程
   - GitHub Actions 使用
   - GitLab CI 使用

3. **实践练习**：
   - 完成每小节后的练习题目
   - 配置 CI/CD 流程
   - 实现自动化部署

## 完成本章后

- 能够理解 CI/CD 的概念和工作流程。
- 能够使用 GitHub Actions 和 GitLab CI。
- 能够实现自动化构建、测试和部署。
