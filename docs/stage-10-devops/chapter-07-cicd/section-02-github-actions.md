## 常见问题

### Q: 如何配置 GitHub Actions 工作流？

A: 在 `.github/workflows/` 目录下创建 YAML 文件，配置工作流。

### Q: GitHub Actions 如何管理密钥？

A: 使用 GitHub Secrets 存储敏感信息，在工作流中引用。
