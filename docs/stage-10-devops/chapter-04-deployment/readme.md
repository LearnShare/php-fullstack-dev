# 10.4 部署选择

## 目标

- 理解不同的部署方式。
- 掌握传统部署和云平台部署的方法。
- 能够根据需求选择合适的部署方式。

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[10.4.1 部署选择](section-01-deployment-options.md)**：部署选择概述、部署方式对比、选型标准、完整示例。

2. **[10.4.2 传统部署](section-02-traditional-deployment.md)**：传统部署概述、部署流程、服务器配置、完整示例。

3. **[10.4.3 云平台部署](section-03-cloud-deployment.md)**：云平台部署概述、主流云平台、部署方法、完整示例。

## 核心概念

- **传统部署**：在物理服务器或虚拟机上部署应用
- **云平台部署**：在云平台上部署应用
- **部署选择**：根据需求选择合适的部署方式

## 学习建议

1. **按顺序学习**：按顺序学习三个小节，理解部署选择的完整体系。

2. **重点掌握**：
   - 部署方式对比
   - 传统部署方法
   - 云平台部署方法

3. **实践练习**：
   - 完成每小节后的练习题目
   - 尝试不同部署方式
   - 对比部署效果

## 完成本章后

- 能够理解不同部署方式的特点。
- 能够进行传统部署和云平台部署。
- 能够根据需求选择合适的部署方式。
