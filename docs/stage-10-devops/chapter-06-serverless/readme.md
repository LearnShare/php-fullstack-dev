# 10.6 Serverless PHP

## 目标

- 理解 Serverless 的概念和优势。
- 掌握 PHP Serverless 的实现方法。
- 能够部署 PHP 应用到 Serverless 平台。

## 章节内容

本章分为两个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[10.6.1 Serverless 概念](section-01-serverless-concept.md)**：Serverless 概述、Serverless 优势、Serverless 平台、完整示例。

2. **[10.6.2 PHP Serverless 实践](section-02-php-serverless.md)**：PHP Serverless 概述、实现方法、部署实践、最佳实践、完整示例。

## 核心概念

- **Serverless**：无服务器计算模式
- **函数即服务（FaaS）**：Serverless 的一种实现方式
- **事件驱动**：Serverless 应用的触发方式

## 学习建议

1. **按顺序学习**：按顺序学习两个小节，理解 Serverless 的完整体系。

2. **重点掌握**：
   - Serverless 概念
   - PHP Serverless 实现
   - 部署方法

3. **实践练习**：
   - 完成每小节后的练习题目
   - 实现 Serverless 函数
   - 部署到 Serverless 平台
