## 常见问题

### Q: GitOps 和传统运维有什么区别？

A: GitOps 使用 Git 管理配置，自动化同步到环境，传统运维手动管理配置。

### Q: GitOps 需要什么工具？

A: 需要 Git 仓库、CI/CD 工具、配置同步工具（如 ArgoCD、Flux）。
