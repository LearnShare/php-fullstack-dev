# 10.9 GitOps

## 目标

- 理解 GitOps 的概念和原理。
- 掌握 GitOps 的实现方法。
- 能够使用 GitOps 管理基础设施和应用。

1. **[10.9.1 GitOps 概念](section-01-gitops-concept.md)**：GitOps 概述、GitOps 原理、GitOps 优势、完整示例。

2. **[10.9.2 GitOps 实践](section-02-gitops-practice.md)**：GitOps 实践概述、工具选择、实现方法、最佳实践、完整示例。

## 核心概念

- **GitOps**：使用 Git 作为单一事实来源的运维方法
- **声明式配置**：描述期望状态而非操作步骤
- **自动化同步**：自动将 Git 中的配置同步到环境

## 学习建议

1. **按顺序学习**：按顺序学习两个小节，理解 GitOps 的完整体系。

2. **重点掌握**：

   - GitOps 实现方法
   - GitOps 工具使用

3. **实践练习**：
   - 完成每小节后的练习题目
   - 实现 GitOps 流程
   - 使用 GitOps 管理应用
