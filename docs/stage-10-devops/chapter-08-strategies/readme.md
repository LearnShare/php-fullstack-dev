# 10.8 Deploy 策略

## 目标

- 理解不同的部署策略。
- 掌握蓝绿部署和金丝雀发布的方法。
- 能够根据需求选择合适的部署策略。

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[10.8.1 部署策略](section-01-deployment-strategies.md)**：部署策略概述、策略类型、策略对比、完整示例。

2. **[10.8.2 蓝绿部署](section-02-blue-green.md)**：蓝绿部署概述、部署流程、实现方法、完整示例。

## 核心概念

- **蓝绿部署**：同时运行两个生产环境，切换流量
- **金丝雀发布**：逐步将流量切换到新版本
- **滚动更新**：逐步更新应用实例

## 学习建议

1. **按顺序学习**：按顺序学习三个小节，理解部署策略的完整体系。

2. **重点掌握**：
   - 部署策略类型
   - 蓝绿部署方法
   - 金丝雀发布方法

3. **实践练习**：
   - 完成每小节后的练习题目
   - 实现蓝绿部署
   - 实现金丝雀发布

## 完成本章后

- 能够理解不同部署策略的特点。
- 能够实现蓝绿部署和金丝雀发布。
- 能够根据需求选择合适的部署策略。
