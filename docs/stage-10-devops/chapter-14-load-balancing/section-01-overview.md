## 常见问题

### Q: 四层和七层负载均衡有什么区别？

A: 四层基于 IP 和端口，七层基于 HTTP 协议，七层更灵活但性能稍低。

### Q: 如何实现会话保持？

A: 使用 IP 哈希、Cookie 等方式实现会话保持。
