## 常见问题

### Q: 如何扫描镜像漏洞？

A: 使用 Docker 安全扫描工具或第三方工具（如 Trivy、Clair）扫描镜像。

### Q: 发现漏洞后如何处理？

A: 更新基础镜像、应用安全补丁、重新构建镜像。
