## 常见问题

### Q: 如何提高容器安全性？

A: 使用非 root 用户、只读文件系统、资源限制、网络隔离等安全措施。

### Q: 容器权限如何控制？

A: 使用 Docker 安全选项（如 `--user`、`--read-only`、`--cap-drop`）控制容器权限。
