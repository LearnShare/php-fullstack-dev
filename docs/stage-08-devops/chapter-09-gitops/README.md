# 8.9 GitOps

## 目标

- 理解 GitOps 的概念与优势。
- 了解 ArgoCD 的使用。
- 掌握基础设施即代码（IaC）。

## 章节内容

本章分为两个独立小节，每节提供详细的概念解释、代码示例和最佳实践：

1. **[GitOps 概念](section-01-gitops-concept.md)**：GitOps 概念、GitOps 原则、GitOps 优势、完整示例。

2. **[GitOps 实践](section-02-gitops-practice.md)**：ArgoCD 使用、应用定义、同步策略、基础设施即代码、完整示例。

## 核心概念

- **GitOps**：Git 作为单一事实来源
- **ArgoCD**：GitOps 工具
- **基础设施即代码**：IaC
- **声明式配置**：声明式部署

## 学习建议

1. **重点掌握**：
   - GitOps 概念
   - ArgoCD 使用
   - 基础设施即代码

2. **实践练习**：
   - 完成每小节后的练习题目
   - 配置 GitOps 流程
   - 实现自动同步

## 完成本章后

- 能够使用 GitOps 管理部署。
- 理解 GitOps 优势，能够建立 GitOps 流程。
- 掌握 ArgoCD，能够实现自动部署。
- 具备构建生产级 GitOps 系统的能力。
