# 8.2 docker-compose 与本地开发环境

## 目标

- 掌握 docker-compose 的配置和使用。
- 能够搭建完整的本地开发环境。
- 理解生产与开发环境的差异配置。

## 章节内容

本章分为两个独立小节，每节提供详细的概念解释、代码示例和最佳实践：

1. **[docker-compose 基础](section-01-compose-basics.md)**：docker-compose 语法、服务定义、网络配置、数据卷、完整示例。

2. **[本地开发环境](section-02-dev-environment.md)**：开发环境配置、热重载、调试配置、多服务编排、完整示例。

## 核心概念

- **docker-compose**：容器编排工具
- **服务编排**：多容器应用管理
- **开发环境**：本地开发配置
- **热重载**：代码变更自动重载

## 学习建议

1. **重点掌握**：
   - docker-compose 配置
   - 服务编排
   - 开发环境搭建

2. **实践练习**：
   - 完成每小节后的练习题目
   - 搭建开发环境
   - 配置多服务应用

## 完成本章后

- 能够使用 docker-compose 管理多容器应用。
- 理解服务编排，能够配置复杂应用。
- 掌握开发环境搭建，能够提升开发效率。
- 具备构建生产级容器应用的能力。
