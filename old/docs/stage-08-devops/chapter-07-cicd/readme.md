# 8.7 CI/CD 与 GitHub Actions

## 目标

- 理解 CI/CD 的概念与流程。
- 掌握 GitHub Actions 的配置。
- 实现自动化测试、构建、部署流程。

## 章节内容

本章分为三个独立小节，每节提供详细的概念解释、代码示例和最佳实践：

1. **[CI/CD 基础](section-01-cicd-basics.md)**：CI/CD 概念、持续集成、持续部署、CI/CD 流程、完整示例。

2. **[GitHub Actions](section-02-github-actions.md)**：GitHub Actions 基础、工作流配置、Actions 市场、完整示例。

3. **[GitLab CI](section-03-gitlab-ci.md)**：GitLab CI 基础、.gitlab-ci.yml 配置、Pipeline 配置、完整示例。

## 核心概念

- **CI/CD**：持续集成和持续部署
- **GitHub Actions**：GitHub 的 CI/CD 平台
- **GitLab CI**：GitLab 的 CI/CD 平台
- **自动化流程**：自动化测试、构建、部署

## 学习建议

1. **重点掌握**：
   - CI/CD 流程设计
   - GitHub Actions 配置
   - 自动化部署

2. **实践练习**：
   - 完成每小节后的练习题目
   - 配置 CI/CD 流程
   - 实现自动化部署

## 完成本章后

- 能够建立完整的 CI/CD 流程。
- 理解 CI/CD 概念，能够设计自动化流程。
- 掌握 CI/CD 工具，能够提升开发效率。
- 具备构建生产级 CI/CD 系统的能力。
