# 8.3 镜像安全

## 目标

- 理解容器镜像的安全风险。
- 掌握 Trivy 等安全扫描工具的使用。
- 了解镜像签名和验证。

## 章节内容

本章分为两个独立小节，每节提供详细的概念解释、代码示例和最佳实践：

1. **[镜像安全](section-01-image-security.md)**：镜像安全风险、安全扫描、漏洞修复、镜像优化、完整示例。

2. **[容器安全](section-02-container-security.md)**：容器运行时安全、权限控制、安全配置、完整示例。

## 核心概念

- **镜像安全**：容器镜像安全
- **安全扫描**：漏洞检测
- **镜像签名**：镜像完整性验证
- **容器安全**：运行时安全

## 学习建议

1. **重点掌握**：
   - 安全扫描工具
   - 镜像签名
   - 容器安全配置

2. **实践练习**：
   - 完成每小节后的练习题目
   - 扫描镜像漏洞
   - 配置安全策略

## 完成本章后

- 能够识别和修复镜像安全问题。
- 理解容器安全，能够配置安全策略。
- 掌握安全扫描，能够持续监控安全。
- 具备构建安全容器应用的能力。
