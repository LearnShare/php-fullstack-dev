# 2.6 表达式与运算符

## 目标

- 掌握算术、比较、逻辑、赋值、位运算等常见运算符。
- 理解运算符优先级、结合性与短路行为。
- 熟练运用三元、空合并、管道式写法提升可读性。
- 掌握 `match` 表达式的使用。

## 章节内容

本章分为六个独立小节，每节提供详细的概念解释、语法说明、参数列表和完整示例：

1. **[算术运算符](section-01-arithmetic-operators.md)**：加法、减法、乘法、除法、取模、指数运算符的语法、用法、类型转换及完整示例。

2. **[赋值运算符](section-02-assignment-operators.md)**：基本赋值、复合赋值、自增自减、数组解构赋值（PHP 7.1+）的语法及使用场景。

3. **[逻辑运算符](section-03-logical-operators.md)**：逻辑与（`&&`）、逻辑或（`||`）、逻辑非（`!`）、`and`、`or`、`xor` 的区别、优先级及短路特性。

4. **[三元运算符与空合并运算符](section-04-ternary-null-coalescing.md)**：三元运算符（`?:`）、省略中间值语法、空合并运算符（`??`）、空合并赋值运算符（`??=`）的用法及优先级。

5. **[match 表达式](section-05-match-expression.md)**：`match` 表达式的语法、与 `switch` 的对比、严格比较、返回值、表达式模式及完整示例。

6. **[运算符优先级与结合性](section-06-operator-precedence.md)**：运算符优先级表、结合性、常见优先级问题、使用括号的最佳实践。

## 运算符分类

| 类型     | 示例                           |
| :------- | :----------------------------- |
| 算术     | `+ - * / % **`                 |
| 赋值     | `=`、`+=`、`-=`、`**=`         |
| 比较     | `==`、`===`、`!=`、`<>`、`<=>` |
| 逻辑     | `&&`、`||`、`!`、`and`、`or`   |
| 字符串   | `.`（连接）、`.=`              |
| 位运算   | `&`、`\|`、`^`、`<<`、`>>`    |
| 其他     | `??`、`?:`、`match`           |

## 学习建议

1. **按顺序学习**：按顺序学习六个小节，理解各种运算符的用法和优先级。

2. **重点掌握**：
   - 算术运算符的基本用法和类型转换
   - 赋值运算符和自增自减的区别
   - 逻辑运算符的短路特性
   - 三元运算符和空合并运算符的使用场景
   - `match` 表达式与 `switch` 的区别
   - 运算符优先级和结合性

3. **实践练习**：
   - 完成每小节后的练习题目
   - 分析现有代码中的运算符使用
   - 使用 `match` 表达式重构 `switch` 语句

4. **理解原理**：
   - 理解运算符优先级和结合性
   - 理解短路求值的原理
   - 理解严格比较和宽松比较的区别

5. **注意陷阱**：
   - 赋值运算符的优先级
   - 逻辑运算符的优先级差异
   - 空合并运算符的优先级

## 完成本章后

- 能够正确使用各种算术、赋值、逻辑运算符。
- 理解运算符优先级和结合性，能够编写正确的表达式。
- 掌握三元运算符和空合并运算符的使用场景。
- 能够使用 `match` 表达式替代 `switch` 语句。
- 理解短路求值的原理，能够优化代码性能。
- 遵循最佳实践，编写清晰、可读的表达式。
