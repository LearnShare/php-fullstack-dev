# 6.3 Secrets 管理

## 目标

- 理解 Secrets 管理的重要性。
- 掌握 `.env` 文件的安全使用。
- 了解 Vault、AWS Secret Manager 等密钥管理服务。
- 熟悉密钥轮换（Key Rotation）的实现。

## 章节内容

本章分为两个独立小节，每节提供详细的概念解释、代码示例和最佳实践：

1. **[Secrets 管理](section-01-secrets-management.md)**：Secrets 管理重要性、`.env` 文件安全、密钥管理服务（Vault、AWS Secret Manager）、密钥轮换、完整示例。

2. **[环境变量与配置](section-02-env-config.md)**：环境变量管理、配置管理、多环境配置、配置验证、完整示例。

## 核心概念

- **Secrets**：敏感配置信息
- **密钥管理**：安全存储和访问密钥
- **环境变量**：配置管理方式
- **密钥轮换**：定期更新密钥

## 学习建议

1. **重点掌握**：
   - Secrets 管理方法
   - 环境变量使用
   - 密钥轮换

2. **实践练习**：
   - 完成每小节后的练习题目
   - 实现 Secrets 管理系统
   - 配置多环境

## 完成本章后

- 能够安全地管理敏感配置。
- 理解密钥管理，能够使用密钥管理服务。
- 掌握环境变量管理，能够配置多环境。
- 具备构建生产级配置管理系统的能力。
