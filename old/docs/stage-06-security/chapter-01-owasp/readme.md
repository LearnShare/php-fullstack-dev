# 6.1 OWASP Top 10（2025）风险模型与防御

## 目标

- 理解 OWASP Top 10 安全风险。
- 掌握 XSS（跨站脚本攻击）的防护方法。
- 了解 CSRF（跨站请求伪造）的防御策略。
- 熟悉 SQL 注入的防护（PDO 预处理 + ORM）。
- 了解其他常见安全风险及防护措施。

## 章节内容

本章分为五个独立小节，每节提供详细的概念解释、代码示例和最佳实践：

1. **[OWASP Top 10 概述](section-01-overview.md)**：OWASP Top 10 概览、安全风险分类、风险评估、完整示例。

2. **[注入攻击防护](section-02-injection.md)**：SQL 注入、XSS 攻击、命令注入、文件包含、防护方法、完整示例。

3. **[认证与会话管理](section-03-authentication.md)**：认证失败、会话管理、密码安全、多因素认证、完整示例。

4. **[其他安全风险](section-04-other-risks.md)**：访问控制失效、安全配置错误、敏感数据暴露、完整示例。

5. **[路径安全](section-05-path-security.md)**：路径遍历攻击、防护措施、输入验证、路径规范化、安全最佳实践。

## 核心概念

- **OWASP Top 10**：十大安全风险
- **注入攻击**：SQL、XSS、命令注入
- **认证安全**：密码、会话、多因素认证
- **访问控制**：权限管理
- **路径安全**：路径遍历攻击防护

## 学习建议

1. **重点掌握**：
   - 注入攻击的防护
   - 认证和会话安全
   - 访问控制
   - 路径安全防护

2. **实践练习**：
   - 完成每小节后的练习题目
   - 实现安全防护机制
   - 进行安全测试

## 完成本章后

- 能够识别和防御常见安全风险。
- 理解 OWASP Top 10，能够实施防护措施。
- 掌握安全最佳实践，能够构建安全应用。
- 具备安全审计和测试的能力。
