# 4.2 HTML 渲染与输出控制

## 目标

- 掌握 PHP 的三种页面渲染方式：纯 PHP 输出、PHP 模板嵌入、模板引擎。
- 理解输出缓冲（Output Buffering）的作用与使用场景。
- 熟悉模板引擎的工作原理，理解为什么模板引擎能提高安全性。
- 能够在实际项目中选择合适的渲染方式。

## 章节内容

本章分为两个独立小节，每节提供详细的概念解释、代码示例和最佳实践：

1. **[HTML 渲染与模板引擎](section-01-rendering-templates.md)**：三种渲染方式对比、PHP 模板嵌入、模板引擎（Twig/Blade）使用、模板继承与组件、安全性（XSS 防护）及完整示例。

2. **[输出缓冲与控制](section-02-output-buffering.md)**：输出缓冲基础、嵌套缓冲、实际应用场景（模板渲染、邮件生成、测试）、性能优化（压缩、缓存）及完整示例。

## 核心概念

- **模板分离**：业务逻辑与视图分离
- **输出缓冲**：控制输出时机和内容
- **自动转义**：防止 XSS 攻击
- **模板继承**：减少重复代码

## 学习建议

1. **重点掌握**：
   - 模板引擎的使用和安全性
   - 输出缓冲的应用场景
   - XSS 防护的最佳实践

2. **实践练习**：
   - 完成每小节后的练习题目
   - 实现一个简单的模板系统
   - 使用模板引擎构建完整页面

## 完成本章后

- 能够选择合适的渲染方式（纯 PHP、模板嵌入、模板引擎）。
- 理解输出缓冲的作用，能够在实际场景中应用。
- 掌握模板引擎的使用，能够构建安全的模板系统。
- 具备防止 XSS 攻击的能力。
